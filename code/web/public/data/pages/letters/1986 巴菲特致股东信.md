# 1986 巴菲特致股东信

## 核心要点

- 伯克希尔净资产增长26.1%，22年复合增长率达23.3%，但未来因规模扩大回报率必然下降
- 优秀经理人是公司成功的关键，管理层的工作就是"不去妨碍他们"
- 资本配置是最大挑战——1986年找不到好的投资机会，只能偿债和囤积现金
- 旗下业务表现出色：[[布法罗新闻报]]渗透率全美第一，[[内布拉斯加家具店]]销售额十年增长三倍，[[喜诗糖果]]扭转了同店销量下滑趋势

## 详细摘要

### 账面价值与内在价值

22年间，每股账面价值从19.46美元增长到2,073.06美元，净资产增长106倍，而流通股份增加不到1%。[[内在价值]]适度超过账面价值，两者比率相当稳定。1986年内在价值增长可能超过账面价值增长，主要归功于关键经理人在几乎不需要额外资本的情况下大幅提升了业务盈利能力——这种未被记录的[[商誉]]价值增长相当可观。

### 管理哲学与经理人

[[芒格]]和巴菲特的两项主要工作：一是吸引并留住优秀经理人，二是资本配置。对于第一项，关键是"不去妨碍他们"——就像管理一支有杰克·尼克劳斯的高尔夫球队，不会对他的挥杆动作指手画脚。

优秀经理人的特质：站在所有者角度思考，对业务充满热忱，工作是因为热爱而非金钱。这种管理风格让[[伯克希尔哈撒韦]]可以轻松扩展业务——当手下都是正直能干的人，可以同时管十几个甚至更多。公司坚持只与喜欢和敬佩的人合作，"天天跟让你反胃的人共事，就像为了钱跟不爱的人结婚一样"。

### 资本配置的挑战

[[伯克希尔哈撒韦]]的资本配置比大多数公司更具挑战性：盈利能力高于平均、收益全部留存、业务只需很少增量资本。1986年收购了菲希海默，但规模只占净资产约2%。公开市场上没有好的投资想法，主要动作是偿还债务和囤积现金。

要实现15%的股本回报率目标，未来十年净资产需增加72亿美元，只有找到几个又大又好的投资机会才有可能。市场终究会发生重大变化，"总有一天又会轮到我们上场"。

### 主要业务表现

**[[布法罗新闻报]]**：斯坦·利普西的超一流管理下，工时大幅下降，成本严格控制。新闻版面比维持在50%，全美同等规模报纸中最高。工作日和周日渗透率均排名全美第一，周日发行量比十年前多出约10万份。

**[[内布拉斯加家具店]]**：[[B夫人]]93岁高龄仍每周七天从开门到打烊在店里工作。净销售额增长10.2%达1.32亿美元，十年前仅4,400万美元。凭借天天低价和最齐全品种，市场辐射范围不断扩大。

**[[喜诗糖果]]**：销售趋势改善，总销售磅数增长约2%，同店销售额基本持平，遏制了此前六年的下滑趋势。查克·哈金斯用顾客满意度衡量成功，这种以客户为导向的精神感染了整个公司。

**[[世界图书百科全书]]**：销量连续第四年增长，百科全书销量比1985年增长7%，比1982年增长45%。超过半数销售人员是现任或前任教师，视自己为教育工作者。

## 原文金句

> "如果我的工作是管理一支高尔夫球队，而杰克·尼克劳斯或阿诺德·帕尔默愿意替我效力，我可不会对他们的挥杆动作指手画脚。"

> "天天跟让你反胃的人共事，就像为了钱跟不爱的人结婚一样——在任何情况下这都不是个好主意，如果你本来就已经很有钱了，那简直就是疯了。"

> "如果我们每个人都雇用比自己矮小的人，我们将变成一家侏儒公司。但如果我们每个人都雇用比自己高大的人，我们将成为一家巨人公司。"

> "市场终究会发生重大变化——这一点可以确定——总有一天又会轮到我们上场。不过我们完全不知道那一天会在什么时候到来。"
