# 1958 巴菲特致合伙人信

## 核心要点

- 1958年股市弥漫"亢奋"情绪，大量投机者涌入，这种"买股票肯定赚钱"的信念预示着麻烦将至
- 合伙人账户平均收益率略高于38.5%，与道指涨幅持平，符合"牛市跟上、熊市跑赢"的预期
- 详细披露联邦信托银行投资案例：以51美元买入[[内在价值]]125美元的股票，最终80美元卖出
- 通过大量买入[[低估]]股票成为大股东，可以主动影响价值修复的时间，而非被动等待

## 详细摘要

### 市场情绪与风险警示

1958年美国股市经历了一场心理大转变，"亢奋"成为最贴切的形容词。无论业余还是专业投资者，都被这股情绪裹挟。巴菲特观察到，越来越多"阴晴不定"的投机者涌入市场，他们能待多久完全取决于能否继续轻松赚快钱。

巴菲特明确表示自己从不试图做[[市场预测]]，精力全放在寻找被[[低估]]的股票上。但他警告：当社会弥漫"买股票肯定赚钱"的信念时，麻烦迟早要来。届时即便是被低估的股票，股价也难免大幅受牵连——尽管[[内在价值]]不会因此减损。

### 业绩回顾与投资哲学

道指全年从435点涨到583点，加上股息，整体涨幅38.5%。五个合伙人账户平均收益率略高于此，各账户介于36.7%到46.2%之间。巴菲特重申其一贯预测：下跌或平盘时跑赢大盘，上涨时能跟上就不错。

### 联邦信托银行案例剖析

巴菲特详细披露了这笔典型投资。联邦信托是新泽西州一家管理上乘的银行，每股盈利约10美元却分文不发股息，导致股价被压在50美元左右，而保守估算的[[内在价值]]为125美元。

投资的[[安全边际]]体现在多个层面：管理优秀、盈利能力强、股价大幅折价、管理层欢迎新股东、亏损风险极小。更关键的是，持股25.5%的大股东多年来一直希望[[收购]]联邦信托，僵局终将打破。

花了约一年时间，巴菲特累积买到12%的股份，平均[[买入价格]]约51美元，成为第二大股东，在任何并购提议上拥有举足轻重的投票权。这只股票每月仅成交两笔左右，股价几乎不受大盘影响。

最终以80美元全部卖出，因为发现了另一个可以成为最大股东的机会。巴菲特强调：内在价值135美元、买价80美元的折价，与内在价值125美元、买价50美元的折价，差距很大。

### 主动创造机会的策略

市场越高，[[低估]]股票越难找。巴菲特正在尝试通过大量买入几只被低估的股票，自己来创造[[套利]]机会。成为大股东的优势在于：可以主动影响价值修复所需的时间，而非被动等待[[市场先生]]的认可。
