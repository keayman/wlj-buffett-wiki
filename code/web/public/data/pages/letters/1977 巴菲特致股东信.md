# 1977 巴菲特致股东信

## 核心要点

- 股本回报率才是衡量管理层经济表现的合理指标，而非每股收益创新高
- 选择在顺风行业而非逆风行业中经营至关重要——保险业即使犯错仍可取得好业绩，纺织业即使管理优秀也只能获得微薄利润
- 通过股票市场购买优秀企业的部分所有权，价格常常大大低于协商收购整家公司的价格
- 保险业务的本质会放大单个管理者对公司业绩的影响，人的因素在此行业确实至关重要
- 对投资决策的评判应依据企业长期经营成果，而非某一天的股价

## 详细摘要

### 经营业绩与回报率思考

1977年经营收益为2,190万美元，折合每股约22.54美元。以期初股东权益计算的经营回报率约为19%，高于长期平均水平及美国企业整体平均水平。巴菲特强调，大多数公司把每股收益创新高称为"创纪录"业绩是没有意义的——股本增加10%而每股收益只增长5%，就像定期存款靠[[复利]]产生越来越多利息一样平淡无奇。[[内在价值]]的真正衡量标准应该是股本回报率。

### 纺织业务的困境

纺织业务表现远低于预期，连续两年的乐观预测都落空。巴菲特坦承继续留在纺织行业的理由：工厂是当地最大雇主，员工年龄偏高且技能难以转移；工人和工会配合改善成本结构；管理层积极坦诚，使纺织部门产出的资金得以调配到利润更高的[[保险业]]收购上。这段经历让巴菲特深刻认识到选择[[护城河]]行业的重要性。

### 保险业务的蓬勃发展

[[伯克希尔哈撒韦]]1967年以约860万美元收购[[国民保险公司]]进军保险行业，当年保费收入约2,200万美元，到1977年已达1.51亿美元——近600%的增长，且未增发任何新股。增长来自传统责任险扩张、新设公司、现金收购及再保险业务。

巴菲特坦诚承认犯过的错误：保证责任险业务、迈阿密城市汽车保险扩张、航空再保险安排等。但保险行业的特性是即使犯错仍可取得不错业绩，这与纺织业形成鲜明对比。

关于[[承保纪律]]，巴菲特指出理赔成本以每月接近1%的幅度增长，源于货币通胀和"社会通胀"。当市场松动、费率不足时，能够坦然接受保费收入减少需要非同寻常的管理纪律——眼睁睁看着别人以愚蠢的价格抢走生意，有悖于一般机构的本能。

### 保险投资与选股标准

保险业务的投资成本从1.346亿美元增长到2.528亿美元，[[保险浮存金]]的增长推动了有价证券投资规模扩大。巴菲特阐述了选股标准，与评估整体[[收购]]企业的方式几乎一样：(1) 能够理解的生意；(2) 良好的长期前景；(3) 诚实且能干的管理层；(4) 价格非常有吸引力。

巴菲特以[[伯克希尔哈撒韦]]自身历史为例说明单一年度快照的局限性：1948年合并前两家公司税后收益近1,800万美元，但合并后十年5.95亿美元销售额换来1,000万美元累计亏损。

投资[[华盛顿邮报]]和[[大都会通信]]等公司时，巴菲特强调不控制反而能获得比控制更好的管理效果——这是非正统但正确的观点。通过股票市场购买优秀企业部分所有权的[[买入价格]]，常常大大低于协商收购整家公司的价格。

### 银行与蓝筹印花

伊利诺伊国民银行资产回报率约为大多数大型银行的三倍，1931年以25万美元创建，1977年收益达360万美元。[[喜诗糖果]]在几乎没有追加资本投入的情况下，税前经营收益从420万美元增长到1,260万美元，在一个几乎没有数量增长的行业中取得这一成绩，展现了[[特许经营权]]的力量。

## 原文金句

> "除非是特殊情况，我们认为股本回报率才是衡量管理层经济表现更合理的指标。"

> "各位的管理层从中学到的一个教训——遗憾的是有时候还得重新学一遍——就是选择在顺风行业而非逆风行业中经营的重要性。"

> "真正出色的企业的部分所有权在证券市场上出售时，其价格常常大大低于通过协商谈判收购整家公司所需要的价格。"

> "不控制反而能获得比控制更好的管理效果。这是一个非正统的观点，但我们认为是正确的。"

> "单一年度的快照作为展现一家企业全貌的工具，大概也不过如此了。"
