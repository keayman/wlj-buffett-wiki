# 1964 巴菲特致合伙人信

## 核心要点

- 1964年合伙基金收益率27.8%，领先道指9.1个百分点，但这是1959年以来领先优势最小的一年——市场急涨时我们本就难以大幅跑赢
- 大型基金公司尽管拥有顶尖人才和充足资源，业绩却普遍跑输无人管理的道指，根本原因在于群体决策、从众倾向、机构惯性等结构性问题
- 真正的保守投资来自正确的前提、准确的事实和合理的逻辑，而非符合常规——市场下跌时的业绩表现才是检验保守与否的客观标准
- 长期目标：预期道指年均收益约7%，合伙基金领先约10个百分点，有限合伙人年均收益约14%

## 详细摘要

### 1964年业绩回顾

合伙基金整体收益率27.8%，道指18.7%，有限合伙人22.3%。领先道指9.1个百分点，是1959年以来最小的领先优势。巴菲特坦言并不沮丧——市场强劲上涨时，合伙基金本就难以大幅跑赢。他甚至庆幸1964年不是被道指远远甩在身后的那一年。

自1957年以来的累计业绩：道指累计收益131.3%，年化11.1%；合伙基金累计608.7%，年化27.7%；有限合伙人累计402.9%，年化22.3%。

### 大型基金为何跑不赢指数

巴菲特将合伙基金与四家最大的基金公司（Massachusetts Investors Trust、Investors Stock Fund、Tri-Continental、Lehman Corporation）进行对比。这四家管理约45亿美元资产，顾问团队年收费超800万美元，但业绩却略逊于无人打理的道指。

他将原因归结为五点：（1）群体决策难以达到一流水平；（2）跟随其他声誉卓著机构的从众倾向；（3）机构框架束缚——做到平均很"安全"，特立独行的风险回报不成比例；（4）僵化固守不理性的[[分散投资]]策略；（5）惯性——这是最重要的原因。

巴菲特犀利指出：任何投资管理机构在对其他公司品头论足之前，都应该先审视自己的方法和业绩。

### 关于保守的真正含义

很多人把符合常规等同于保守，巴菲特认为这是错误的。真正的保守投资来自正确的前提、准确的事实和合理的逻辑。民意调查替代不了独立思考。

如何客观评估投资策略是否保守？研究市场下跌时的业绩表现。在1957、1960、1962三个下跌年份，有限合伙人累计收益+45%，而四大基金和道指均为负收益（-9.7%至-24.6%）。

### [[复利]]的威力

巴菲特用曼哈顿岛交易的故事说明复利的惊人力量：1626年印第安人以24美元卖出曼哈顿，若按6.5%年收益率复利338年，将变成421亿美元，远超当时曼哈顿约124亿美元的土地价值。

财富积累的三个有利因素：寿命长、[[复利]]增长率高、两者兼得。年化收益率提高几个百分点，最终结果差距惊人——这正是合伙基金追求超越平均收益的根源。

### 长期目标与预期

巴菲特坦承过去27.7%的收益率无法长期持续，原因包括：资金规模膨胀的限制、股票整体重估带来的高收益不可持续、领先优势必然波动。

他的长期预期：道指年均约7%（波动剧烈）；合伙基金领先道指约10个百分点；有限合伙人年均约14%。他强调这只是猜测，但认为所有投资管理人都应开诚布公地说明目标和衡量标准。

## 原文金句

> "真正的保守投资，来自正确的前提、准确的事实和合理的逻辑。遵循这三点做出的决定，有时与常规相符，但更多时候是与常规背道而驰的。"

> "民意调查替代不了独立思考。我们真正释然一笑的时候，是遇上一个我们能看懂的机会——事实清楚明了，该怎么做一眼就能看出来。"

> "有个人去面试，说自己有20年从业经验，他前任老板纠正说——不是'20年的从业经验'，而是'1年的经验，重复了20遍'。"

> "受不了热，就别进厨房。如果你持有的股票市值下跌20%或30%，你就会遭受情感创伤或陷入财务困境，那你就不适合投资股票。"

> "预测往往能让人看清预测者的弱点，却未必能看清未来。"
