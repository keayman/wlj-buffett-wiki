# 1969年5月 巴菲特致合伙人信

## 核心要点

- 正式宣布将结束巴菲特合伙基金，因为投资环境恶化且个人希望改变生活方式
- 定量分析能找到的投资机会在过去二十年稳步减少，现在几乎找不到了
- 承诺为不想自己打理资产的合伙人推荐可靠的资产管理人
- 合伙人可自由选择获得现金或有价证券，包括[[伯克希尔哈撒韦]]等控股公司股份
- 坦承今年套利投资失利是自己的评估失误，而非运气问题

## 详细摘要

### 投资环境的恶化

巴菲特指出当前投资环境面临三重困境：通过定量分析能找到的[[低估]]机会在过去二十年稳步减少，现在几乎绝迹；合伙基金1亿美元的规模限制了投资选择，300万美元以下的投资对整体业绩无意义；市场上追逐业绩的人越来越多，[[市场先生]]的眼光越来越短，投机氛围日益浓厚。

### 退出的个人原因

巴菲特坦言，尽管18个月前已调低业绩目标，但只要"在场上"，定期公开业绩，承担为合伙人管理资产的责任，他就无法真正放松。他直言："我不想自己一辈子都在比拼投资，都在和一只兔子赛跑。要慢下来只有一个办法：结束。"

### 清算安排与合伙人选择

巴菲特承诺为合伙人推荐一位人品和能力俱佳的资产管理人，无账户最低金额限制。清算时，合伙人可自由选择获得现金或有价证券。对于[[伯克希尔哈撒韦]]、多元零售公司等控股公司股份，合伙人可按权益比例获得，但这些股票受SEC限制，短期内不能自由买卖。巴菲特特别提到新[[收购]]的伊利诺伊国民银行，称其为"经营特别出色的银行"。

### 坦承投资失误

巴菲特直言今年[[套利]]投资"特别不顺"，自比"误入羽毛球场的一只小鸟"。他强调这完全是自己的评估失误，而非运气问题——他误判了政府趋势的发展变化。他反思道：做决策时不应把"应该怎样"和"实际会怎样"混为一谈。

### 对未来的态度

巴菲特表示不想以业绩惨淡的一年谢幕，但看不到任何机会扭转局面，也不愿拿别人的钱"误打误撞碰运气"。关于未来打算，他坦言没有答案，只知道60岁时应该有不一样的追求，需要与现在的事业"一刀两断"才能适应新生活。

## 原文金句

> "我不想自己一辈子都在比拼投资，都在和一只兔子赛跑。要慢下来只有一个办法：结束。"

> "谁都不愿意在白纸黑字上把自己的错误写出来，但是我看不起报喜不报忧的行为。"

> "做决策的时候，根本不应该把自己认为应该怎样（对社会有益）和实际会怎样混为一谈。"

> "我不想为了辉煌谢幕而做自己不懂的投资，不想毁了这么多年的好业绩。"

> "有些东西是20岁的我非常想要的，但当我60岁的时候，自己应该有不一样的追求。"
