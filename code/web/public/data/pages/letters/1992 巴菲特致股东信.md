# 1992 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]每股账面价值增长20.3%，28年来年复合增长率约23.6%，但未来资本规模将成为相对业绩的拖累
- 发行新股的铁律：除非收到的价值不低于付出的价值，否则绝不发行——目标是同时增加股东财富和扩大规模
- [[收购]]哲学的转变：从以便宜价格买普通公司，转向以合理价格买好公司——"亲吻蟾蜍"的教训
- 对股市未来十年回报持谨慎态度，但强调衡量标准是长打率而非打击率，接受并欢迎短期波动
- [[透视收益]]比GAAP数字更能反映真实盈利，2000年目标是从6.04亿增长到18亿美元

## 详细摘要

### 业绩与股票发行原则

1992年[[伯克希尔哈撒韦]]净资产增加15.2亿美元，超过98%来自经营收益与投资增值。因赎回[[可转换证券]]债券，部分持有人选择转股，公司发行了6,106股新股。

关于发行新股，巴菲特立下铁律：必须确保收到的价值不低于付出的价值。他用一个银行并购的故事说明"扩大规模"与"增加股东财富"并不总是一致——被收购方老板甚至要求承诺"以后不许再做这么蠢的交易"。

### 业绩波动与市场预测

由于会计准则要求保险公司持有的股票按市价计算，[[伯克希尔哈撒韦]]的年度业绩波动较大。巴菲特指出三个评估要点：旗下非股市相关业务的贡献逐年变化；企业税对投资收益的侵蚀是结构性劣势；未来十年标普500回报将远不如过去十年。

[[芒格]]和巴菲特虽然一贯认为[[市场预测]]是"毒药"，但他们确信股市表现不可能永远超越其背后的企业。他们接受并欢迎波动，因为对短期波动的容忍能改善长期前景。

### 所罗门事件收尾

巴菲特担任所罗门公司临时董事长十个月后卸任。在他缺席期间，[[伯克希尔哈撒韦]]经营未受影响。所罗门在《财富》杂志调查中声誉改善程度排名第二，证券子公司税前收益创历史新高。

### 收购哲学的演变

巴菲特坦承早年也"跟蟾蜍约过会"——试图以便宜价格买普通公司。他用童话比喻那些花重金亲吻"企业蟾蜍"期待变王子的经理人，最终只能宣布"重组"减记，学费由股东买单。

职业高尔夫教练的建议让他顿悟："练习无法造就完美，练习只会巩固习惯。"于是他改变策略，开始以合理[[买入价格]]买进好公司。

1992年收购了中央州保险公司82%股权，由老友比尔·凯泽管理。这笔交易与26年前收购[[国民保险公司]]有诸多相似——都是奥马哈公司，都是老友主动找上门。

### 透视收益与长期目标

[[透视收益]]包含报告收益、被投资公司留存收益中应占份额、减去估计税负。1992年透视收益6.04亿美元，要实现15%年增长目标，2000年需达到18亿美元。

巴菲特强调长期目标不意味着忽略短期成果。他引用爱丽丝的话警告那些长期无法兑现承诺的管理者："总该有些果酱是今天就能吃到的吧！"

### 保险行业状况

1992年行业综合比率约114.8%，其中约4个百分点归因于安德鲁飓风——有史以来最大单次保险赔付损失，摧毁了几家小型保险公司。
