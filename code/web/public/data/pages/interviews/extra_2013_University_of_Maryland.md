# extra_2013_University_of_Maryland

## 核心要点

- 规模是业绩的敌人：管理2100亿美元的回报率必然低于管理100万美元，但巴菲特仍然选择前者
- 投资风格演变：从[[格雷厄姆]]的定量"捡烟蒂"策略，逐渐转向[[芒格]]和费雪的"以合理价格买入优秀企业"
- 年度股东信的写作原则：把股东当作合伙人，用写给不懂金融的姐妹的方式解释公司情况
- 2008年金融危机的关键时刻：雷曼倒闭引发货币市场基金挤兑，美国工业几乎停摆

## 详细摘要

### 关于投资门槛与规模

[[伯克希尔哈撒韦]]没有设定固定的投资门槛收益率。巴菲特坦言，资本规模增大使得获取超额回报变得更加困难。如果管理100万美元，回报率会比管理2100亿美元（伯克希尔净资产）更高。规模是业绩的敌人，但他仍然宁愿管理2100亿美元。

### 投资哲学的演变

巴菲特在哥伦比亚大学师从[[格雷厄姆]]，形成了早期投资策略。格雷厄姆的方法非常量化，长期来看不会亏钱，但随着资金规模增大，适用性越来越有限。

"捡烟蒂"策略就像在地上找到一根恶心的雪茄，还剩一口可以抽，捡起来点着，免费抽一口。巴菲特早年就是这样寻找极度便宜的股票。

在接触费雪和[[芒格]]之后，他开始寻找更好的公司。现在的策略是寻找优秀企业，而非仅仅便宜的企业。[[BNSF铁路]]就是典型案例——收购前税前利润30亿美元，现在已达60亿美元，而且10年、100年后铁路依然会是好生意。

巴菲特分享了两个"捡烟蒂"的实例：1951年发现西部保险公司每股盈利29.09美元，股价却只有16美元；2004年通过花旗的韩国股票手册，一个下午找到约20家市盈率仅2倍的公司。"如果你买20只市盈率2倍的股票，你一定会赚钱。"

[[芒格]]曾对他说："你永远不会和我意见不同，因为你很聪明，而我总是对的。"

### 股东信的写作哲学

巴菲特把股东视为合伙人，思考如果角色互换，他希望了解什么信息。初稿会写"亲爱的姐妹们"，用她们能理解的方式解释。每年还会写一个与[[伯克希尔哈撒韦]]不直接相关的投资教学章节，2013年的信中有2600字（总共11500字）是关于投资思考的内容。

1956年他用0.49美元买了一本账簿，用两页纸写合伙协议，只用半页纸解释基本规则：能做什么、不能做什么、如何衡量成功。"如果你觉得合适就加入，不合适我们还是朋友。"这就像婚姻，在重要问题上必须有共识。

### 2008年金融危机与特殊投资

高盛和通用电气在2008年向[[伯克希尔哈撒韦]]寻求融资支持，这是巴菲特从未想过会发生的事。伯克希尔获得了优先股和认股权证，后来通过协商结算，不用支付现金就获得了近20亿美元的高盛股票。目前仍持有美国银行的认股权证，可在2021年8月前以每股7.14美元购买7亿股。

雷曼倒闭导致货币市场基金"跌破面值"，引发了对3.5万亿美元货币市场基金的电子挤兑，三天内流出1750亿美元。商业票据市场冻结，美国工业几乎停摆。布什总统说："如果资金不松动，这个国家就完了"——巴菲特认为这是有史以来最伟大的经济声明。
