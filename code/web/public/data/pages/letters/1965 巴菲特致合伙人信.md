# 1965 巴菲特致合伙人信

## 核心要点

- 合伙基金取得47.2%的收益率，领先道指33个百分点，创下成立以来最大相对优势
- 资金规模已达4364.5万美元，开始对未来业绩产生潜在不利影响，决定停止接受新合伙人
- 取得[[伯克希尔哈撒韦]]控股权，从"低估类"投资转为"控制类"投资
- 强调业绩衡量标准的重要性，批评基金行业普遍缺乏明确的评估体系
- 各类投资中，"低估类（相对低估）"重要性日益增加，但稳妥程度不如其他类别

## 详细摘要

### 年度业绩回顾

1965年合伙基金实现47.2%的收益率，扣除普通合伙人分成后有限合伙人获得36.9%回报，而道指整体收益率仅14.2%。这是基金成立以来取得的最大相对优势，年末资产较年初增加1230万美元。

巴菲特坦言这一业绩偏离了"每年领先指数10个百分点"的目标，并幽默地表示"惭愧"。他提醒合伙人，投资不像通用汽车那样有用户忠诚度和品牌惯性，每年都要从零开始，过去的方法和机会无法延续到未来。

### 基金行业业绩对比

巴菲特将合伙基金与四家大型基金公司（Massachusetts Investors Trust、Investors Stock Fund、Lehman、Tri-Continental）进行对比。这些公司管理着50多亿美元资产，每年收取1000多万管理费，但九年累计业绩均未能战胜道指。

他引用弗兰克·布洛克的观点，批评投资行业普遍缺乏业绩衡量体系："为别人管理资产，却不知道自己的表现如何，我们的过失同样不可原谅。"

### 规模与未来预期

资金规模达到4364.5万美元后，巴菲特首次表示规模可能开始产生不利影响。过去几年，适中的规模反而有利——某些投资机会正好适合当时的体量。但现在，规模拖累业绩的可能性已大于提升业绩的可能性。

基于此判断，他决定停止接受新合伙人，并强调这一政策必须一视同仁——即使自己再有子女想加入，也要另寻投资途径。现有合伙人的追加投资暂不受限。

### 各类投资进展

**[[套利]]类**：机会零星，平均投入600万美元，实现141万美元收益，回报率23.5%。其中一半以上来自单笔交易。

**[[低估]]类（基于产业资本视角）**：贡献丰厚收益。一笔通过要约收购卖出的投资实现318.8万美元收益。这类投资的逻辑是：产业资本愿意按某价格收购控股权，但散户却不愿按同样价格买入少量股份。

**控制类**：新增[[伯克希尔哈撒韦]]。从1962年11月开始以每股7.60美元买入，1965年春取得控股权，年末净营运资本已达每股19美元。公司虽处于生存压力巨大的纺织行业，但拥有一流经理人肯·切斯。巴菲特表示控股公司的价值取决于公司本身，股价短期波动与持有决策无关。

**低估类（相对低估）**：随着其他机会减少，重要性日益增加。巴菲特坦言这类投资"有些虚浮"，不如其他类别稳妥，但1965年表现优异，主要得益于两笔投资。
