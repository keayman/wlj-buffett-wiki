# 1963年中 巴菲特致合伙人信

## 核心要点

- 上半年合伙基金上涨14%，道指上涨10%，但短期业绩意义有限，尤其对含控股类投资的基金而言
- 不同投资类别在不同市况下表现迥异，但绝不根据市场预测调整仓位配置
- [[登普斯特农具机械制造公司]]在哈里·博特尔管理下实现质变，每股估值从35.25美元升至64.81美元
- 合伙基金存在的根本意义是在较低风险下实现超越大盘的复利增长

## 详细摘要

### 上半年业绩与投资哲学

1963年上半年，道指整体收益率为10%，合伙基金（不含登普斯特估值变化）上涨14%。巴菲特再次强调两点：三年以下的短期业绩意义不大；市场下跌时我们通常能跑赢，但市场泡沫高涨时跟上基准已属不易。

[[低估]]类投资上半年净投资约527.5万美元，收益率约21%，表现出色。而[[套利]]类在上涨市场中拖累了整体业绩——这是牛市中的正常表现。巴菲特明确表示，从不靠猜测[[市场预测|市场走势]]来调整仓位，无论是直接猜市场，还是间接猜哪类投资近期最占优势。

### 与基金公司的业绩对比

巴菲特列出了合伙基金与四家大型基金公司的累计业绩对比。从1957年至1963年6月30日，合伙基金累计收益355.8%，年化[[复利]]收益率26.3%；而道指累计77.8%，年化9.3%。收费高昂的机构投资建议始终难以跟上无人主动管理的蓝筹股指数。

这些投资渠道为投资者提供了[[分散投资|充分分散风险]]、省心方便等服务，但不包括让资金以超越大盘的速度复利增长。合伙基金存在的根本意义，就是在承担较低长期本金损失风险的前提下，实现更高的复利增长速度。

### 登普斯特的华丽转身

哈里·博特尔被巴菲特称为"年度之星"还说轻了。经过19个月的努力，[[登普斯特农具机械制造公司]]的资产负债表发生质变：

- 1961年11月：调整后每股价值35.25美元
- 1962年11月：调整后每股价值51.26美元
- 1963年6月：调整后每股价值64.81美元

哈里将资产变现所得的价格远超巴菲特的保守估值。公司主营业务在大幅精简资产后具备了适度的盈利能力。巴菲特用一个有趣的细节说明[[内在价值]]：合伙基金持有登普斯特71.7%股份，成本126万美元，而公司保险柜里的证券市值中对应份额就有145万美元——公司土地上的所有东西全是白赚的利润。

### 预存资金与税务考量

合伙基金接受预存资金，按6%计息；也允许合伙人提前取款（上限20%），同样按6%计息。虽然银行贷款利率更低，但预存资金可预期将成为权益资本，投资心态与短期借款截然不同。

关于[[税收效率|税收]]，巴菲特旗帜鲜明地主张多交所得税——但要享受低税率。太多投资决策拿"税收考量"当借口。投资决策应以税后净资产在最低风险下实现最高复利为出发点，把低成本股票单独隔离另眼对待，无异于让投资管理形同虚设。

## 原文金句

> "我从不靠猜测市场走势来调整仓位。从长期来看，我们认为试图判断市场涨跌是一门很差的生意，我们不玩这个。"

> "合伙基金存在的根本意义，就是与上述投资渠道相比，在承担较低长期本金损失风险的前提下，实现更高的复利增长速度。"

> "我的净资产等于持仓市值减去出售时应缴的税款。这笔税负跟资产一样实实在在，一分都少不了——除非资产价值下跌（疼啊），或者把资产捐出去，再或者把它带到棺材里。"

> "和那些虔诚地等待有人以明年盈利35倍来收购相比，我的'数手指头脚趾头'方法更有底气得多。"
