# 1968年11月 巴菲特致合伙人信

## 核心要点

- 合伙基金严格不接受新合伙人，无一例外，包括现有合伙人的家属
- 控股权益估值应基于资产价值和盈利能力，而非市场价格
- 今年业绩远超预期，领先道指超过15个百分点，但1969年尚无可用投资机会

## 详细摘要

### 年度承诺书与税务安排

巴菲特提醒合伙人尽早返还1969年承诺书，12月31日后不可更改。税收情况与8月信中一致，但年底前可能有重大交易影响纳税，届时会在12月24日信中更新。

### 控股权益估值方法

合伙基金持有多元零售公司80%股份，估值从初始成本480万美元升至880万美元，采用"成本加[[留存收益]]"方法。

持有[[伯克希尔哈撒韦]]70%股份，估值从每股25美元提升至31美元。巴菲特强调：市场价格对控股权益估值意义不大，真正的决定因素是资产价值和盈利能力。去年估值高于市价，今年估值低于市价，恰好说明市场价格是次要的。

### 严格的合伙人准入政策

合伙基金坚决不接受新合伙人，包括现有合伙人的配偶、子女、孙辈。巴菲特解释：允许新成员加入会损害现有合伙人利益，且他不愿区别对待任何人。合伙权益也不可转移至信托，因为这违反合伙性质的税收规定和合同约定。

### 业绩表现与未来展望

今年业绩超出预期。尽管大部分时间持有大量现金等价物，有价值的机会寥寥无几，但仍领先道指超过15个百分点（道指上涨约8%）。然而，1969年可用的投资机会目前一个都没有。

## 原文金句

> "市场价格支配的是少数股东权益的价值，对控股权益估值没多大意义。在对控股权益估值时，主要决定因素是资产价值和盈利能力。"

> "允许新成员加入，会损害所有现有合伙人的利益，而且我也不愿意区别对待各位合伙人，答应这个，不答应那个。"

> "我们能在1969年能用得上的投资机会现在还一个都没有。"
