# 2024年伯克希尔股东大会

## 核心要点

- [[苹果]]仍将是伯克希尔最大持仓，卖出部分股票是为增加现金储备，而非对公司看法改变
- 伯克希尔净资产达5710亿美元，稳居全球第一，现金储备接近2000亿美元
- 投资的核心原则：买股票就是买公司，不预测市场，将市场为我所用
- 伯克希尔以纳税为荣，预计未来税率可能上升，当前21%的税率或许是较好的卖出时机

## 详细摘要

### 一季度业绩概览

2024年第一季度[[伯克希尔哈撒韦]]经营利润达112亿美元，远高于去年同期。[[保险业]]表现出色，但[[阿吉特·贾恩]]提醒不能简单乘以四估算全年——飓风季节通常在三季度带来最大风险。[[保险浮存金]]的投资收益因利率上升而大幅增长。[[BNSF铁路]]利润略有下降，与行业运量下滑相关，但利润水平仍有改善空间。

### 资本配置与现金储备

截至一季度末，伯克希尔持有1820亿美元现金和短期国债，二季度末预计接近2000亿美元。巴菲特表示想把钱投出去，但必须满足两个条件：风险非常低，且能赚很多钱。当前股价水平下，[[回购]]难以为股东创造显著价值，主要因为伯克希尔股东多为长期投资者，交易量有限。

### 投资哲学的阐述

巴菲特回顾了自己的投资历程：早年走了很多弯路，直到读到[[格雷厄姆]]的《聪明的投资者》才顿悟——用买公司的眼光买股票，不被[[市场先生]]牵着鼻子走。后来遇到[[芒格]]，在其启发下认识到买好生意的重要性。[[美国运通]]、[[可口可乐]]、[[苹果]]都是按这个原则买入的好生意。

### 关于卖出苹果股票

卖出[[苹果]]并非因为对公司看法改变，而是基于当前股市估值和世界局势，增加现金储备更为有利。巴菲特特别提到税收考量：现行21%的联邦企业所得税率可能会上升，美国财政赤字难以持续，政府迟早要提高税收。

### 关于中国投资

伯克希尔的主要投资将始终在美国。投资[[美国运通]]、[[可口可乐]]这样的美国公司，实际上已经获得了全球业务的敞口。
