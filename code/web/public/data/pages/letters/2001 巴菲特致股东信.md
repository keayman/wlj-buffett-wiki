# 2001 巴菲特致股东信

## 核心要点

- 2001年[[伯克希尔哈撒韦]]净资产减少37.7亿美元，每股账面价值下降6.2%，但仍跑赢标普500指数5.7个百分点
- 完成多项重要收购（萧氏工业、佳斯迈威、迈迪科、XTRA、乐森卓尔），全部以现金支付，不稀释股东权益
- 严厉批评企业高管通过股票期权损害股东利益的行为，强调管理层与股东利益一致的重要性
- 坦承个人投资组合管理表现不佳，以及允许通用再保险在缺乏安全边际情况下承保的错误

## 详细摘要

### 业绩回顾与投资哲学

[[伯克希尔哈撒韦]]2001年每股账面价值下降6.2%，但相对标普500指数的-11.9%仍领先5.7个百分点。自1965年以来，公司年复合增长率达22.6%，累计回报194,936%。巴菲特强调相对业绩的重要性——只要能持续跑赢大盘，长期回报必然可观。

巴菲特坦言未来难以复制过去的辉煌：一是优质企业和证券的[[买入价格]]已远高于从前；二是资金规模太大，需要"大象级别"的猎物才能产生实质性增长。1973年投资[[华盛顿邮报]]或1976年投资盖可保险那样的机会，如今即便有十个也只能让净资产增加0.25%。

### 公司治理与管理层激励

巴菲特严厉抨击当时盛行的[[公司治理]]乱象：股东蒙受数十亿损失，而CEO们却通过股票期权带着巨额财富全身而退。他用一个辛辣的笑话讽刺期权重新定价的恶习。

他承诺自己99%以上的身家都在[[伯克希尔哈撒韦]]，从未卖过一股，与股东利益完全一致。他和[[芒格]]对那些把股东当"冤大头"而非合伙人的企业领导人深恶痛绝。

### 2001年收购活动

**迈迪科**：全球领先的屋顶桁架连接板制造商。55名管理团队成员以真金白银购买了10%股权，每人至少出资10万美元。巴菲特强调这才是真正的企业家精神——承担下跌风险，享受上涨收益，无法"重新定价"。

**XTRA**：货柜拖车租赁业领导企业。交易原定9月11日完成，尽管有退出条款，[[伯克希尔哈撒韦]]仍如期履约。

**乐森卓尔**：美国定制画框行业龙头，CEO克雷格·庞齐奥将公司从300万美元做到3亿美元。从首次通话到达成协议仅用两天。

**鲜果布衣**：巴菲特回顾了1955年在[[格雷厄姆]]-纽曼公司工作时参与收购联合内衣公司的往事——那笔交易以极低成本获得了巨额回报。如今鲜果布衣虽已破产，但在约翰·霍兰德的整顿下正在恢复元气。

### 威利家具的扩张奇迹

[[威利家居]]在坚持周日不营业的"反常规"政策下，成功进军博伊西和拉斯维加斯。亨德森新店营业额打破所有连锁店记录，是巴菲特预期的两倍。巴菲特自嘲：如今每当他对零售业高谈阔论，伯克希尔的人只会问"那Bill怎么看？"

### 保险业务的核心逻辑

巴菲特阐述评估[[保险业]]公司的三个关键因素：[[保险浮存金]]规模、浮存金成本、以及这两个因素的长期前景。浮存金是保险公司持有但不归其所有的资金，可用于投资，这是[[伯克希尔哈撒韦]]的核心竞争优势之一。
