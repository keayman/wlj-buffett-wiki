# extra_1998_buffett_florida

## 核心要点

- 成功的关键不在于智商和精力，而在于诚信、品格和行为习惯——这些都是可以培养的
- "雪茄烟蒂"式投资法虽然有效，但时间是优秀企业的朋友、糟糕企业的敌人
- 日本企业股本回报率低（4%-6%），即使借贷成本只有1%，也很难找到好的投资机会
- 低回报的企业即使买得便宜，长期持有也只能获得糟糕的结果

## 详细摘要

### 关于成功的品质

[[巴菲特]]向MBA学生提出一个思想实验：如果你能买入一位同学未来收入的10%，你会选谁？你不会选IQ最高或成绩最好的人，而是会选择那个慷慨、诚实、能领导他人、愿意把功劳归于别人的人。

反过来，如果要做空一位同学，你会选那个自负、贪婪、走捷径、不诚实的人。关键在于：左边那些令人钦佩的品质都是可以习得的，右边那些令人厌恶的品质都是可以摒弃的。

习惯的锁链起初轻得感觉不到，直到重得无法打破。年轻时改变习惯比年老时容易得多。[[格雷厄姆]]年轻时就观察他钦佩的人，然后模仿他们的行为。

### 关于日本市场

[[伯克希尔哈撒韦]]可以在日本以1%的利率借入10年期贷款。理论上只需要赚取超过1%的回报就能盈利，但至今没有找到合适的投资标的。

日本企业的[[内在价值]]问题在于股本回报率极低，只有4%-6%。当企业本身赚不到多少钱时，投资者很难获得高回报。日本曾经有令人难以置信的市场，却没有令人难以置信的公司——它们业务量大，但股本回报率不高，这最终反噬了它们。

### 关于"雪茄烟蒂"投资法

这是一种非常便宜的、量化的投资方法：寻找股价远低于营运资本的股票。就像在街上找到一个潮湿、令人厌恶的雪茄烟蒂，但还能免费抽一口。这种方法有效，但不优雅。

然而，时间是优秀[[商业模式]]的朋友，是糟糕生意的敌人。如果长期持有糟糕的生意，即使[[买入价格]]很低，结果也会很糟糕。如果长期持有优秀的生意，即使买入时稍微贵了一点，结果也会很好。
