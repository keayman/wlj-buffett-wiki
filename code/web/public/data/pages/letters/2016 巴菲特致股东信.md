# 2016 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]52年来每股账面价值年复合增长率约19%，但随着业务重心从证券投资转向全资控股企业，[[内在价值]]与账面价值的差距持续扩大
- 公司坚持用现金而非股票进行[[收购]]，1998年用股票收购通用再保险是重大错误，导致流通股增加21.8%
- 美国经济体系创造了奇迹般的财富积累，长期持有美国优质企业的投资者几乎肯定会获得不错回报
- [[回购]]只在股价低于[[内在价值]]时才对继续持有的股东有意义，价格是关键考量因素

## 详细摘要

### 账面价值与内在价值的脱节

2016年[[伯克希尔哈撒韦]]净资产增加275亿美元，每股账面价值增长10.7%。52年来每股账面价值从19美元增至172,108美元。

早期公司资源主要配置在有价证券上，账面价值与[[内在价值]]相近。但1990年代初重心转向全资控股企业后，会计准则的不对称处理——"输家"必须减记而"赢家"永不上调——导致两者差距扩大。如今赢家企业中巨大的未记录收益，使股票[[内在价值]]远超账面价值，这在[[保险业]]中尤为明显。

### 业务转型的关键历程

[[芒格]]和我预期公司每股盈利能力每年都会增长。作为[[留存收益]]金额连续两年位列美国第一的公司，这些再投资资金必须赚回自己的存在价值。

转型过程中有几个关键节点：1993年以4.34亿美元收购[[德克斯特鞋业]]是严重错误，用股票支付的25,203股到2016年底价值已超60亿美元。1996年全现金收购GEICO剩余股权是正确之举，它成为公司核心。1998年用股票收购通用再保险使流通股增加21.8%，是另一个重大失误。2000年现金收购[[中美能源]]76%股权让我们走上正轨。

自1999年以来，公司流通股仅增长8.3%，大部分来自收购[[BNSF铁路]]——那是一次有意义的股票发行。

### 对美国经济的信心

美国240年来将人类智慧、[[市场先生|市场体系]]、移民和法治融合，创造了先辈做梦都想不到的丰饶，累积了90万亿美元财富。

20世纪道琼斯指数从66点涨到11,497点，实现17,320%的资本收益。美国企业未来几年几乎肯定会比现在值钱得多。创新、生产力提升、企业家精神和充裕资本将确保这一点。

市场偶尔会出现严重下跌甚至恐慌，但广泛的恐惧是投资者的朋友，因为它会送来便宜货。避免高昂费用、长期持有大型财务保守的美国企业的投资者，几乎肯定会做得不错。

### 回购的正确理解

对退出股东而言，[[回购]]总是有利的。但对继续持有的股东，回购只在股价低于[[内在价值]]时才有意义。令人费解的是，企业宣布回购时几乎从不提及价格上限。

即使股价被[[低估]]，两种情况下也不应回购：一是企业需要资金保护或扩展运营；二是有更好的[[收购]]机会。

[[伯克希尔哈撒韦]]被授权在账面价值120%或更低时大量回购，因为这个价位相对[[内在价值]]有很大折让。我们更愿意看到股价在[[内在价值]]附近窄幅波动。
