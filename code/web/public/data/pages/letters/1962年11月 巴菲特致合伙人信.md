# 1962年11月 巴菲特致合伙人信

## 核心要点

- 截至10月底，合伙基金收益率5.5%，超越道指22.3个百分点，创历史最大领先幅度
- [[登普斯特风车制造公司]]资产变现成效显著：存货从420万降至190万美元，现金从16.6万增至约100万美元
- 明确警告合伙人：过去几年的优异业绩属于异常，未来领先幅度将大幅收窄
- 业绩优势60%来自一般投资组合，40%来自登普斯特估值提升
- 强调不做[[市场预测]]，资金配置是被低估类股票价格过高"逼"出来的

## 详细摘要

### 年度行政事务

信件开篇处理例行文书工作：修订合伙协议、增资提现承诺书等。协议实质内容未变，仅纳入新合伙人并改写部分条款使其更清晰。提现款1月1日后立即拨付，增资款须在1月10日前到账，提前汇入按6%年息计算。

巴菲特特别重申：月度支付绝非保证收益，只是便利的定期提现安排——收益超6%则来自盈利，不足则部分或全部是本金返还。

### 登普斯特的资产变现奇迹

1962年的核心成就是将[[登普斯特风车制造公司]]大量资产几乎以原价变现。按去年的折扣估值法（存货六折、应收账款八五折、固定资产按拍卖价），去年底每股估值约35美元，今年底至少应达50美元。

财务状况的改善堪称戏剧性：1961年11月底，公司仅有16.6万美元现金却背负231.5万美元负债；预计今年底将拥有约100万美元现金及投资，负债仅剩25万美元。1963年有望继续改善，投资组合规模将大幅扩张。

### 业绩归因与预期管理

截至10月底，道指下跌约16.8%，合伙基金收益5.5%，领先22.3个百分点。巴菲特坦言这一优势主要因为在道指大跌时，资金大量配置在控制类和[[套利]]类而非一般市场仓位。

他反复强调：这是异常业绩，重演机会极少。如果道指当年大涨，相对业绩反而会很难看。资金配置并非源于[[市场预测]]能力，而是因为[[低估]]类股票价格太贵，被迫转向其他类别。合伙人必须做好心理准备——肯定会有跑输道指的年份。

## 原文金句

> "上述业绩纯属异常，日后能重演的机会极少。"

> "今年迄今的好成绩，绝对不是因为我能猜准市场走势（我从来不猜），只不过是因为低估类股票的价格当时太贵，逼着我不得不把更多资金放到其他类别。"

> "我们充分做好了心理准备——肯定会有跑输道指的年份。我当然不认为这种情况会长期存在，否则我早就认输，直接买道指了。"

> "请各位不要以为过去几年的业绩还会继续重演。日后我们领先道指的幅度将会小得多。"
