# 2011年巴菲特与南京大学EMBA学生畅言谈投资与人生全中文纪录

## 核心要点

- 投资决策的核心在于判断企业未来的经济前景，而非纠缠于法律细节或尽职调查
- 寻找投资标的的三个条件：能理解的生意、可信任的管理层、合理的价格
- 好的生意如同经济城堡，需要不断加宽的[[护城河]]和优秀的管理者守护
- 灾难事件（如日本地震）不会改变长期投资决策，经济具有自我恢复能力
- 估值不依赖Excel表格，而是基于对企业经济特性的深刻理解

## 详细摘要

### 收购哲学：简单与信任

[[巴菲特]]展示了1983年收购[[内布拉斯加家具店]]的协议，仅4页纸，没有审计报告和律师参与。他强调投资犯错的风险在于错误计算公司未来的经济前景，而非法律细节。以德克斯特鞋业为例，错误判断美国制鞋业前景导致4亿美金损失归零，这比环保或租赁协议问题严重得多。

### 灾难与经济韧性

谈及日本地震，[[巴菲特]]认为这不会改变其购买股票或生意的决策。灾后重建会拉动经济，3年后日本经济可能比现在更强。他指出经营[[保险业]]的经验告诉他：所有可能发生的最终都会发生。但此事件可能对美国核电产业前景产生深远影响。

### 投资标准的一致性

无论在中国还是美国，投资标准一致：

第一，能够理解这个生意，对5年、10年、20年后的经济前景做出有确定性的评估，充分了解产业竞争动态和公司的竞争位置。

第二，找到有能力且可信任的管理层。[[伯克希尔哈撒韦]]总部人员精简，不会派MBA到处运营，必须买到生意的同时买到管理团队。

第三，需要一个与未来经济前景相比合乎道理的[[买入价格]]。

### 护城河理论

[[巴菲特]]将好的生意比作强大的经济城堡。以[[可口可乐]]为例，总有人想推出产品打败它，因此需要优秀的骑士警惕守护。公司的每一个行为，不管大小，不是在加强就是在削弱这条[[护城河]]。[[内布拉斯加家具店]]从8000万美金增长到单店4亿美金，正是护城河不断加宽的体现。

### 中石油投资案例

[[巴菲特]]分享了投资[[中国石油]]的经历：仅通过阅读年报判断公司价值约1000亿美金，而市值仅350亿美金。他强调先形成价值判断再看价格，避免市场报价影响估值。中石油承诺将45%利润分红并信守承诺，这在大型石油公司中极为罕见。最终选择中石油而非俄罗斯尤科斯，是因为对中国的信心更强。

### 金融危机的反思

2008年金融危机是全球经济的急刹车，冲击比大萧条初期更猛烈。[[巴菲特]]感谢伯南克、保尔森和布什政府的迅速行动。他指出当大众恐惧时会群体恐慌，经济活动迅速冻结，但解冻需要时间。政府拥有让经济恢复的工具，私营经济体没有这个能力。

### 估值方法论

[[巴菲特]]从不用Excel表格计算估值，认为详细的数据表格会给人虚假的安全感，让人误以为未来更有规律。他不看公司管理层的业绩预测，因为每个公司都会预测明年赚更多。真正的估值是在头脑中对企业经济前景形成判断，然后问：这个价格能否带来期待的长期回报？资金成本就是能找到的次好投资机会。

## 原文金句

> "认为犯错误的风险在于错误计算了公司未来的经济前景，而不是纠缠于法律的细节。"

> "在经济世界中，我把好的生意看成一个强大的经济城堡，公司的每一个行为，不管大小，不是在加强就是在削弱这条护城河。"

> "我一般是先看报告，形成对于公司价值的判断，然后再看公司的价格，否则先看公司的价格会潜意识的影响对于公司价值的判断。"

> "我不喜欢任何可能传达虚假假设的计算。通过在excel表中列出详细的数据表格，人们会获得一种虚假的安全感，让人们认为未来会更加的有规律，但是事实并不是这样。"

> "当我发出火警警报的时候，可以让你们每个人都快速的冲向门口，但我不能让你们以同样迅猛的回到座位上。"
