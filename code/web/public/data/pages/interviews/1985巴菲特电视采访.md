# 1985巴菲特电视采访

## 核心要点

- 投资的核心规则：不要赔钱，以远低于价值的价格买入
- 成功投资需要的是稳定的性格，而非高智商
- 远离华尔街的喧嚣，专注于企业本身的价值而非股价波动
- 坚守能力圈，不懂的东西坚决不碰
- 耐心等待最佳机会，可以数年不出手

## 详细摘要

### 投资的基本原则

巴菲特开门见山地阐述了投资的两条规则：第一条是不要赔钱，第二条是不要忘记第一条。他强调，如果你以远低于[[内在价值]]的[[买入价格]]购入一组资产，基本上就不会亏钱。这体现了[[格雷厄姆]]价值投资的核心理念——[[安全边际]]。

### 投资者的性格品质

巴菲特认为，成功投资需要的是性格品质而非智力品质。你不需要很高的智商，不需要会下三维象棋。关键是要有稳定的性格——既不因随大流而快乐，也不因逆大流而快乐。正如[[格雷厄姆]]所说：你正确与否，不取决于有多少人同意或反对你，而取决于你的论据和推理是否正确。

### 远离华尔街的智慧

巴菲特选择留在奥马哈而非华尔街，因为在华尔街会受到过度刺激，听到太多噪音，导致短视。在奥马哈，他能获取事实而非刺激，专注于企业本身的价值。他不需要身处华盛顿才能评估[[华盛顿邮报]]的价值——这只是一个理性思考的过程，干扰越少越好。

### 能力圈与耐心

巴菲特强调要定义自己的能力圈，在圈内找出相对于价值售价最便宜的标的。他坦言从未买过[[科技与互联网]]公司，包括[[IBM]]这样的伟大公司，因为他不懂。他认为不必在每场游戏中都赚钱。

在证券市场，没有"被判好球"一说。你可以看着成千上万个机会飘过，直到等到一个正中下怀的球再挥棒。他可能两年都不出手，这让大多数人感到无聊，但正是这种耐心成就了他。

### 为何简单方法不流行

巴菲特的方法看似简单，却少有人效仿。学术界热衷于研究各种变量——周二买周五卖是否更赚钱、选举年买入是否更好。但这些都不重要。一个商人购买企业时不会考虑这些，购买股票时为何要考虑？因为股票只是企业的一部分。
