# 1991 巴菲特致股东信

## 核心要点

- 伯克希尔净资产增长39.6%，但规模扩大意味着未来难以维持过去的增长速度
- [[可口可乐]]和[[吉列]]贡献了21亿美元增长中的16亿，但这种估值重估不可能年年发生
- 媒体行业正从特许经营权向普通企业转变，估值逻辑发生根本性改变
- [[喜诗糖果]]收购二十年的成功案例展示了[[特许经营权]]的威力：700万净资产创造了4.1亿税前利润
- 透视收益下降14%，主要受媒体业下滑、[[富国银行]]盈利疲软等因素影响

## 详细摘要

### 规模与增长的困境

1991年[[伯克希尔哈撒韦]]净资产增加21亿美元，增幅39.6%。自现任管理层接手27年来，每股账面价值从19美元增长到6,437美元，年[[复利]]增长率约23.7%。

但巴菲特坦言：随着公司规模扩大到74亿美元，能显著影响业绩的投资机会持续缩小。过去管理2,000万时，赚100万就能增加5个百分点回报；如今需要3.7亿美元才能达到同样效果。他和[[芒格]]设定了[[内在价值]]年均增长15%的目标，未来十年需赚220亿美元。

### 可口可乐与吉列的双重收益

去年账面价值的超额增长源于[[可口可乐]]和[[吉列]][[市盈率]]大幅飙升。三年前买入可口可乐时，伯克希尔净资产约34亿；现在仅可口可乐持股市值就超过这个数字。

巴菲特称这两家是"当今世界上最好的两家公司"，预期盈利将强劲增长。但去年获得了"双重收益"——盈利增长加上市场价值重估——这种情况不可能年年发生，未来只能靠单一收益。

### 所罗门事件与管理层信任

1991年8月，巴菲特被选为所罗门公司临时董事长。他愿意接受这个职位，隐含着一个重要信息：伯克希尔旗下经理人如此出色，他可以大幅减少花在公司上的时间。布鲁姆金家族、查克·哈金斯等都是各自领域的顶尖高手，根本不需要他的帮忙。

他的工作只是公平对待他们，并配置他们创造的资金。被问及何时退休，他回答："大概在我死后五到十年吧。"

### 透视收益的概念与应用

透视收益包括：报告收益、被投资公司未反映在利润中的[[留存收益]]、以及假设分配时需缴纳的税负。1991年透视收益下降14%，原因包括：[[媒体与出版]]收益下降、[[吉列]]优先股被强制转换、[[富国银行]]基本损益两平、保险业务收益下滑。

巴菲特建议投资者也关注自己的透视收益，建立一个十年后能产生最高透视收益的投资组合。这种方法迫使投资者思考企业长期前景而非股价短期表现。"投资就像打棒球，想要在计分板上添分，眼睛得盯着球场，而不是盯着计分板。"

### 媒体业的结构性转变

媒体公司盈利能力下降既有周期性因素，也有长期趋势性因素。随着零售业态转变、广告和娱乐选择多元化，曾经风光无限的媒体企业[[特许经营权]]正在被侵蚀。

巴菲特区分了两类企业：[[特许经营权]]企业的产品被需要、无近似替代品、不受价格管制，能容忍糟糕管理；普通企业只有作为低成本运营商或供不应求时才能获得超额利润，可以被糟糕管理毁掉。

估值数学的变化令人警醒：过去媒体资产被视为6%永续增长的年金，100万收益可值2,500万；若变成周期性波动的普通企业，同样收益只值1,000万。[[华盛顿邮报]]和[[大都会通信]]因强健的资产负债表和优秀管理，仍是最让人安心的媒体投资。

### 喜诗糖果二十年

1972年蓝筹印花以2,500万美元收购[[喜诗糖果]]（扣除多余现金后的实际价格）。当时公司有形净资产仅700万，查理和巴菲特还没充分认识[[特许经营权]]的价值。

二十年后成绩斐然：销售额从2,900万增至1.96亿，税前利润从420万增至4,240万。更惊人的是，公司目前只需2,500万净资产运营，仅追加了1,800万留存收益，其余4.1亿税前利润全部分配出来用于再投资。

收购时的关键洞见是：这家企业拥有尚未开发的[[买入价格]]提价能力。查克·哈金斯被任命全面掌管，薪酬安排只用五分钟握手敲定，从未签过书面合同，沿用至今。
