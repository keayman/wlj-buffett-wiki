# 1966 巴菲特致股东信

## 核心要点

- 1966年公司财务实力恢复到1960年底水平，账面价值周转率回归正常
- 通过[[回购]]计划，流通股减少37%，每股账面价值从23.37美元升至28.99美元
- 保持健康财务状况是核心目标，为应对周期波动、潜在[[收购]]和技术变革做好准备
- 公司开始将部分营运资金投资于普通股，以追求更高回报并间接参与纺织业以外的利润

## 详细摘要

### 1966年运营状况

本年度营业总额4,940万美元，与去年持平但产品结构变化较大。合成部门销售下滑，被家居纺织品部门增长所弥补。下半年市场整体低迷，面临染纱进口冲击、流行趋势转变、产能过剩等多重压力。公司主动关停产线避免库存积压，截至信件撰写时纺织市场仍未回暖。

### 六年运营回顾

对于纺织这样高度周期性的行业，[[伯克希尔哈撒韦]]过去十年不过是盈利期与亏损期交替上演的老剧本。1966年格外重要——不仅实现盈利，更见证公司财务实力恢复。

六年间股息支付超过净收益，但账面价值下降22%的主要原因是[[回购]]计划持续买回股票。流通股缩减至1,017,547股，较1960年减少37%。每股账面价值从23.37美元升至28.99美元，充分体现了[[回购]]对现有股东的好处。

### 财务状况的维护

保持健康财务状况始终是核心目标，使公司得以在周期性行业中存活至今。截至年末营运资金达2,316万美元，每股营运资金从14.41美元升至22.76美元。

维持强健财务有几层考虑：公司积极寻找[[收购]]标的但尚无成功案例；当前货币市场难以为[[收购]]融资，充足流动资产至关重要；战争风险、税率变动等不确定因素也强调了这一必要性。

家居纺织品部门增长迅猛，过去三年销售额几近翻番，未来可能需要追加高达700万美元的投资。技术变革威胁如影随形，重要降本设备问世时若赶上行业低迷，外部融资极为困难，充足营运资金格外关键。

公司年末在营运资本中保留540万美元有价证券，包括短期市政[[债券]]、商业票据和普通股。董事会认为应对这部分资本积极追求合理回报，将大部分暂时投资于普通股，有望获得更高收益并间接参与纺织业务以外的利润。
