# 1987 巴菲特致股东信

## 核心要点

- 伯克希尔七大非金融业务以1.75亿美元股本创造约1亿美元税后净利，股本回报率高达57%，且几乎不使用杠杆
- 最好的商业回报来自那些今天做的事情和五年、十年前大致相同的公司——剧烈变动和出色回报通常不会同时出现
- 对旗下经理人最大的贡献是适时给予掌声，而非过度干预日常运营
- 真正优秀的企业通常不需要借钱，充分发挥已有的强大[[特许经营权]]才是创造非凡经济效益的秘诀

## 详细摘要

### 账面价值与[[内在价值]]的关系

1987年伯克希尔净资产增加4.64亿美元，增幅19.5%。自现任管理层接手23年来，每股账面价值由19.46美元增长到2,477.47美元，年[[复利]]增长率约23.1%。

账面价值和[[内在价值]]之间常常毫无关联。LTV和鲍德温联合公司破产前账面价值分别为6.52亿和3.97亿美元，而贝尔里奇石油公司账面价值仅1.77亿美元却以36亿美元出售。在[[伯克希尔哈撒韦]]，这两种价值走势相当接近，过去十年[[内在价值]]增长率略超账面价值增长率。

### 七大非金融业务的卓越表现

[[布法罗新闻报]]、菲希海默制服、柯比吸尘器、[[内布拉斯加家具店]]、斯科特费泽制造集团、[[喜诗糖果]]和[[世界图书百科全书]]七家企业1987年息税前利润高达1.8亿美元，净利息支出仅200万美元。以1.75亿美元历史成本股本计算，税后净利约1亿美元，股本回报率约57%。

《财富》杂志报道，1000家最大企业中过去十年平均股本回报率超过30%的仅有6家，表现最好的也不过40.2%。这七家企业由于经营所需资本极少，可以将几乎全部收益释放出来投入新机会。

### 经理人的价值与管理哲学

CEO们很少承认公司由笨蛋经营，年报中常见惨淡业绩后却称管理团队是"公司最宝贵的资产"。在[[伯克希尔哈撒韦]]，对经理人的评价是低调的——布卢姆金家族、海德曼家族、查克·哈金斯、斯坦·利普西和拉尔夫·谢伊，每位都将卓越才能、充沛精力和正直品格融为一体。

[[芒格]]和巴菲特在日常运营上没什么好插手的。没有公司会议，没有年度预算，也没有绩效考核。对旗下子公司最大的贡献是适时给予掌声——基于数十年密切观察企业经营所做出的真诚肯定。

### 稳定性与[[特许经营权]]的重要性

投资者往往把最高[[市盈率]]给那些许诺翻天覆地变化的公司，但经验表明最好的商业回报来自今天做的事情和五年、十年前大致相同的公司。不断遭遇重大变化的企业会面临许多犯重大错误的机会，在永远剧烈变化的领域中很难建立堡垒般的[[特许经营权]]。

《财富》杂志研究显示，1977-1986年间满足十年平均股本回报率超过20%且没有任何一年低于15%的仅25家公司。这些冠军大多杠杆很低，所在行业相当平凡，销售的产品与十年前基本相同。

### 各业务近况

[[内布拉斯加家具店]]由94岁的[[B夫人]]掌舵，1987年净销售额1.426亿美元，增长8%。她每天从开门干到打烊，一周七天。客户通过购买NFM产品省下约3000万美元。

[[布法罗新闻报]]在渗透率和新闻版面占比两项指标上领跑全国。新闻版面占比保持在50%左右，若削减到40%每年可节省400万美元新闻纸成本，但对此毫无兴趣。

[[喜诗糖果]]1987年销量创下近2500万磅新高，约85%的利润在12月单月创造。查克·哈金斯百分之百以客户为导向，当淘汰的口味引发客户抗议时，不仅恢复了产品，还把失误变成商机。
