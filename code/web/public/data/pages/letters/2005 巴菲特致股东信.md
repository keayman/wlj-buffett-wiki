# 2005 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]的价值由两大要素构成：每股投资金额和非保险业务的每股税前收益，两者在过去40年分别实现了28%和17.2%的复合增长率
- 2005年完成五项重大[[收购]]，全部以现金完成，不涉及发行股票——这一点至关重要，因为用股票收购实际上是在出售现有股东的权益
- 尽管遭受34亿美元飓风损失，[[保险浮存金]]成本仍为零，这要归功于GEICO等业务的卓越表现
- [[中美能源]]收购太平洋电力公司，将新增160万电力用户，公用事业领域的大额投资能为伯克希尔的巨额资金提供合理回报
- 面对飓风频率和强度的不确定性，采用帕斯卡式的风险管理思维——在无知面前选择谨慎

## 详细摘要

### 估算[[内在价值]]的框架

[[巴菲特]]开篇即阐明报告目的：为股东提供估算[[伯克希尔哈撒韦]][[内在价值]]所需的信息。他坦承内在价值计算"不可避免地不够精确"，但伯克希尔因拥有多种稳定收益来源、充裕流动性和极低负债，估值精度高于大多数公司。

公司将68种业务归为四大类分别讨论。[[巴菲特]]提出一个关键问题：伯克希尔作为母公司究竟给子公司带来了什么好处？股东直接持有这些企业是否更有利？

### 2005年[[收购]]活动

这一年完成五项收购，全部以现金支付：

医疗防护公司（MedPro）是一家106年历史的医疗事故保险公司。[[巴菲特]]强调[[承保纪律]]高于一切，而伯克希尔的财务实力能让医生安心。

森林河公司的收购展现了伯克希尔独特的并购风格——从收到传真到握手成交仅用一周。创始人皮特曾把公司卖给杠杆收购公司，被指手画脚后离开，公司随即破产。皮特买回公司后，选择了"绝不会告诉他该怎么经营"的伯克希尔。

美国商业资讯公司的收购源于CEO凯西读到《华尔街日报》文章后的一封信。[[巴菲特]]特别欣赏她信中的话："我们把公司管理得很紧凑，没有秘书，也没有管理层级。"

[[中美能源]]收购太平洋电力公司需34亿美元，将新增160万电力用户。[[巴菲特]]指出公用事业虽无超高利润，但能为大额资金提供合理回报。

### [[保险业]]的核心逻辑

[[保险浮存金]]从1967年的2,000万美元增长到490亿美元。当取得承保利润时，浮存金成本甚至低于零——"拿着别人的钱还要别人倒贴给我们"。

GEICO保单增长12.1%，市场份额升至6.1%。广告支出从1996年的3,100万美元增至5.02亿美元。[[巴菲特]]回忆21岁时发现GEICO的简单公式：低成本带来低价格，低价格带来稳定增长的盈利性业务。

面对飓风风险的不确定性，[[巴菲特]]引用帕斯卡的赌注：既然不知道答案，就应该选择谨慎。他们决定只在远高于去年的价格水平上承保超级巨灾保险。

关于保险准备金，[[巴菲特]]分享了一条惨痛教训："保险业的意外远不是对称的。每十个坏消息中能遇到一个好消息，你就算走运了。"

## 原文金句

> "当一家公司的管理层得意洋洋地用股票收购另一家公司时，收购方的股东们同时也在出售自己在所有现有资产中的一部分权益。"

> "浮存金是件美妙的事——前提是获取成本不能太高。"

> "保险业的意外远不是对称的。每十个坏消息中能遇到一个好消息，你就算走运了。"

> "与许多并购方不同，伯克希尔没有'退出策略'。我们买入就是为了永久持有。"
