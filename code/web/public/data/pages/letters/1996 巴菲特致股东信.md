# 1996 巴菲特致股东信

## 核心要点

- 净资产增加62亿美元，增幅36.1%，32年来每股账面价值年复合增长率约23.8%
- 完成两笔重要收购：堪萨斯银行家保险公司和飞安国际公司，后者约15亿美元
- 保险浮存金规模达67亿美元，且连续四年实现承保利润，资金成本为负
- 股价与内在价值的关系趋于合理，强调长期持有的重要性

## 详细摘要

### 业绩表现与价值衡量

1996年[[伯克希尔哈撒韦]]净资产增加62亿美元，每股账面价值增长31.8%。由于发行新股收购飞安国际及发售B级普通股，流通股数有所增加。自现任管理层接手32年来，每股账面价值从19美元增长至19,011美元，年[[复利]]增长率约23.8%。

[[芒格]]和巴菲特一再强调，真正重要的是[[内在价值]]而非账面价值。他们提供了两项关键指标：每股投资金额（含现金）和每股税前经营收益。1996年每股投资金额增长29%，非投资本业收益增长63.2%。

### 成本控制的重要性

伯克希尔税后总部费用占净资产比率不到万分之二。相比之下，股票型共同基金每年约100个基点的费用可能吃掉投资者10%甚至更多的回报。"我们是来跟各位一起赚钱的，不是来赚各位的钱的。"

### 股价与内在价值的关系

1995年时巴菲特曾警告股价涨幅超过[[内在价值]]增长。1996年情况逆转：内在价值大幅增加，股价却几乎没变，价格/价值关系回归合理。长期而言，股东整体收益必然与企业经营收益一致。管理层应通过政策和沟通促进市场价格与内在价值同步。

### 1996年的收购

**堪萨斯银行家保险公司**：一家专注银行家保险的公司，拥有出色承保记录。收购源于一次生日宴会上的偶然交谈，从接触到成交仅用了不到一个月。

**飞安国际公司**：全球飞行员训练领导者，收购金额约15亿美元。创始人阿尔·乌尔斯基79岁仍像55岁般精力充沛。公司在41个地点运营175台飞行模拟器，属于资本密集型产业。交易结构鼓励对税务无差异的股东选择现金，最终约51%换成现金，49%换成伯克希尔股票。

### 保险业务的卓越表现

[[保险浮存金]]是衡量保险业务的关键。浮存金是持有但不属于公司的资金，产生于保费在损失理赔前收取。1996年浮存金规模达67亿美元，自1967年以来年复合增长率22.3%。连续四年实现承保利润，意味着资金成本为负——不但不用花钱持有这些资金，反而因持有它们而获得回报。

### 巨灾再保险业务

巨灾险业务大多数年份赚大钱，偶尔录得巨亏。1996年承接了几笔大型风险敞口：与好事达保险签约承保佛罗里达飓风险，可能是有史以来单一公司独力承担的最大单笔巨灾风险；为加州地震局签出保单，潜在赔付金额超过佛罗里达保单一倍以上。

巴菲特强调：保险业遭遇特别糟糕年头不是可能性，而是确定性。"查理跟我宁可要波动的15%长期回报，也不要平稳的12%。"
