# 2007年央视专访巴菲特

## 核心要点

- 投资股票的本质是投资公司，要看企业未来五到十年的发展，而非短期价格波动
- 价格越高越要谨慎，巴菲特在油价30美元时对[[中国石油]]很乐观，75美元时就持中性态度
- 不懂的东西不投资，宁可错过999只股票，也要确保选中的那只是对的
- 赚钱不是明天或下周的问题，而是买入五年十年能升值的东西
- 财富积累到一定程度后应回馈社会，巴菲特将80%财产捐赠慈善

## 详细摘要

### 中石油投资的得与失

巴菲特2003年以约1.6至1.7港元买入[[中国石油]]H股23.4亿股，投入约5亿美元，2007年全部清仓时赚了40亿美元。他专门给中石油写了感谢信，称其记录比世界上任何石油企业都好。

关于为何卖出，巴菲特解释：石油公司的利润主要依赖油价。30美元一桶时价格很有吸引力，他非常肯定；到75美元时就持中性态度。这体现了他对[[买入价格]]和[[安全边际]]的重视。

### 价值投资的精髓

巴菲特阐述了[[内在价值]]投资的核心：看企业本身，看公司未来五年十年的发展，了解公司业务，信任管理层，在价格合适时持有。他强调这"简单但不容易"，需要稳定的情感和态度，而非巨大的智慧。

他从不做投机，持有[[可口可乐]]长达20年、美国运通14年。他希望投资者把自己想象成公司所有者，而非持有一纸价格每天变动的凭证。

### 对中国市场的看法

巴菲特认为中国有很多好企业，但股市发展强劲后，已不像两年前那么容易找到被[[低估]]的股票。面对A股45%股票[[市盈率]]超过100倍的现象，他建议要谨慎——任何东西巨幅上涨时，人们容易被表象迷惑。

### 投资方法论

巴菲特每年阅读成千上万份年报，买入中石油前仅凭2002和2003年年报就决定投资5亿美元，没见过管理层，也没看分析师报告。他强调"你必须知道你买的是什么"，不会因宏观因素改变投资策略——即使美联储说明年经济衰退，找到好企业他还是要买。

### 财富观与生活方式

巴菲特将370亿美元（财产的80%）捐赠慈善，他和太太50年前就达成共识：财富足够家庭富裕生活后，应回馈社会。他住在奥马哈小镇最普通的房子里，保持着简单朴素的生活方式。

## 原文金句

> "投资的精髓，不管你是看公司还是股票，要看企业的本身，看这个公司将来五年十年的发展，看你对公司的业务了解多少，看管理层是否喜欢并且信任，如果股票价格合适你就持有。"

> "我说简单但是不容易，不需要巨大的智慧，一般的智慧就可以，但很多时候需要一种稳定的情感和态度。"

> "赚钱不是明天或者下个星期的问题，而是你是买一种五年或十年的时候能够升值的东西。"

> "如果有1000个股票，对999我都不知道，我只选那只我了解的。"

> "希望你不要认为自己拥有的股票仅仅是一纸价格每天都在变动的凭证，相反，我希望你将自己想象成为公司的所有者之一。"
