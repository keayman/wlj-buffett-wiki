# 1984 巴菲特致股东信

## 核心要点

- 1984年净资产增长13.6%，低于过去22年22.1%的年复合增长率，资本规模膨胀必然拖累回报率
- 股票回购在价格远低于[[内在价值]]时对股东极为有利，但"绿邮讹诈"式回购则是对无辜股东的剥削
- [[内布拉斯加家具店]]凭借超凡运营效率（运营费用仅占销售额16.5%）创造了美国单店最高销售纪录
- 寻找符合定性和定量标准的投资机会十多年来从未如此困难，"无为而治"说起来容易做起来难

## 详细摘要

### 业绩回顾与增长预期

1984年[[伯克希尔哈撒韦]]净资产增加1.526亿美元，每股增长133美元，增长率为13.6%。虽然数字听起来不错，但必须与产生它的资本相比较。过去20年每股账面价值年复合增长率为22.1%，但这一增长率已成历史。要在未来十年达到15%的年均增长率，需要赚取约39亿美元，这需要几个大点子——小打小闹不行。[[芒格]]和巴菲特目前还没有好主意，但根据经验，好点子偶尔会冒出来。

### 特殊股息交易的会计处理

[[伯克希尔哈撒韦]]与盖可保险和通用食品进行了按比例赎回交易：在这些公司回购股票时，同步出售相应股份以保持持股比例不变。这类交易在税务上享受6.9%的低税率股息待遇。然而，毕马威纽约总部不同意将其列为股息，要求作为股票出售处理。这仅是会计差异，对[[内在价值]]没有任何影响。

### 股票回购的价值

当经营出色、财务稳健的公司发现股价远低于[[内在价值]]时，[[回购]]是最能让股东受益的举措。[[回购]]的好处有二：一是简单的算术——以远低于每股内在商业价值的价格回购，花1美元能获得2美元现值；二是管理层明确表明致力于增加股东财富而非扩张势力范围，这会提升市场对公司的估值。

巴菲特严厉批评"绿邮讹诈"式[[回购]]：勒索者持股墨迹未干就发出通牒，管理层用无辜股东的钱息事宁人，最终无辜股东默默埋单。

### 内布拉斯加家具店的卓越表现

[[B夫人]]91岁仍担任董事长，每周七天从开门工作到关门。她一天做出的决策可能比大多数CEO一年做的还多。5月她获得纽约大学商学荣誉博士学位——此前一天学都没上过。

[[内布拉斯加家具店]]净销售额达1.15亿美元，是美国单一家居用品店的最高纪录。其成功秘诀在于超凡效率：毛利率不到行业龙头莱维茨的一半，运营费用仅占销售额16.5%（莱维茨为35.6%）。布鲁姆金家族的经营哲学：对工作充满热情、务实界定能力圈、拒绝能力圈外的诱惑、保持最高道德品行。

收购时[[伯克希尔哈撒韦]]没有要求审计、没有盘点存货、没有核验产权——给了[[B夫人]]5,500万美元支票，她给了承诺。这是一笔公平的交易。

### 喜诗糖果经营状况

[[喜诗糖果]]1984年销售收入1.36亿美元，税后利润1,338万美元，销售糖果2,476万磅，门店数达214家。

### 投资组合

主要持仓包括盖可保险（市值3.97亿美元）、通用食品（2.26亿美元）、埃克森石油（1.75亿美元）、[[华盛顿邮报]]（1.50亿美元）等。寻找符合标准的投资机会十多年来从未如此困难。

## 原文金句

> "经济收益必须与产生它的资本相比较来评估。"

> "当经营出色、财务稳健的公司发现自己的股价远低于内在价值时，没有什么比回购自家股票更能让股东受益的了。"

> "一个在回购明显有利于股东的情况下却始终拒绝回购的管理层，暴露出来的动机远比他自己意识到的要多。"

> "我们给了B夫人一张5,500万美元的支票，她给了我们她的承诺。这是一笔公平的交易。"

> "便宜卖，讲实话。"
