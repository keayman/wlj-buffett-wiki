# 1976 巴菲特致股东信

## 核心要点

- 经营收益创纪录达1607.3万美元，股本回报率17.3%，[[保险业]]承保业务超预期复苏是主要驱动力
- 纺织业务令人失望，但仍承诺继续经营——它是当地重要就业来源，长期可获合理回报
- 投资组合高度集中于少数精选标的，选股标准与100%[[收购]]企业一致：好生意、好管理、好价格、能力圈内
- 伊利诺伊国民银行平均资产收益率约2%，远超同业，成本控制是关键

## 详细摘要

### 整体业绩与展望

1976年经营收益达每股16.47美元，创历史新高。但[[巴菲特]]强调股本回报率才是衡量经济表现更有意义的尺度——17.3%虽高于美国工业平均水平，却仍低于1972年的19.8%历史纪录。预计1977年美元收益可能增长，但回报率将略有下滑。

### 纺织业务的困境

纺织部门无论销售利润率还是资本回报率都差强人意。新收购的Waumbec纺织厂营销与产能未能匹配，设备和人员判断失误导致成本偏差。尽管如此，公司仍在寻找扩张机会——纺织业虽难以提供高回报，但作为新贝德福德和曼彻斯特地区的重要雇主，值得长期承诺。

### 保险承保的强劲复苏

[[国民保险公司]]传统车险和一般责任险业务表现优异，综合比率从115.4大幅降至98.7，远优于行业平均。菲尔·利舍的领导功不可没。但[[巴菲特]]警告：业务高度集中于受通胀冲击严重的险种，今天充足的费率明天未必够用。预计1978年前后，非理性竞争将使费率再度落后于成本，届时必须大幅削减承保量。

再保险业务改善滞后，定价失误的影响持续时间更长。"本土州"业务在约翰·林沃特管理下取得实质进展，保费增长78%。家庭和汽车保险公司在约翰·苏厄德领导下强劲复苏。

### 投资策略与持仓

税前投资收益增至1082万美元。债券市场反弹使组合呈现未实现净收益，但[[巴菲特]]表示更愿意看到债券价值低于账面——这样新资金能获得更高收益率。

股票组合坐拥4570万美元未实现收益。主要持仓包括盖可保险、[[华盛顿邮报]]、凯撒工业等。[[巴菲特]]阐述了集中投资的逻辑：选股标准与100%[[收购]]企业一致——（1）良好的长期经济特质；（2）称职诚实的管理层；（3）以私有化标准衡量具有吸引力的[[买入价格]]；（4）能力圈内的行业。符合条件的机会稀少，因此集中持有少数标的。

### 银行业务的卓越表现

伊利诺伊国民银行平均资产收益率约2%，比号称"主要银行中最好"的克利夫兰国民城市公司高出近50%。这一成绩是在支付最高存款利率、保持极佳流动性、回避低质量贷款的条件下实现的。成本控制是关键——尽管业务大幅扩张，员工人数仍维持在1969年收购时的水平。

## 原文金句

> "我们认为股本回报率才是衡量经济表现更有意义的尺度。"

> "今天充足的费率，明天未必还够用。"

> "我们的投资选择立足长期，所考量的因素与100%收购一家企业时并无二致。"

> "能够同时满足这些条件的投资机会并不多见，这也是我们集中持股的原因之一。我们实在找不到一百只符合我们投资要求的股票。"

> "只要公司的业务成果在数年内持续优秀，无论市场价值年复一年如何大幅波动，我们终将从股票持仓中获得良好的财务回报——对此我们深信不疑。"
