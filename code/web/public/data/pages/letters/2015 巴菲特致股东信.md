# 2015 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]净资产增长154亿美元，51年来每股账面价值年复合增长率约19.2%，但[[内在价值]]已远超账面价值
- [[BNSF铁路]]投入创纪录58亿美元资本支出改善服务，税前收益达68亿美元创新高
- 以320亿美元现金[[收购]]精密铸件公司(PCC)，将成为"六驾马车"新成员
- [[保险业]]连续第13年实现承保盈利，[[保险浮存金]]从410亿美元增至880亿美元
- 对美国经济前景保持乐观，认为今天出生的美国婴儿是有史以来最幸运的一代

## 详细摘要

### 账面价值与内在价值的差距

[[伯克希尔哈撒韦]]2015年每股账面价值增长6.4%。早期公司资源主要配置在有价证券上，账面价值与[[内在价值]]大致相当。但1990年代初转向完全持有企业后，会计准则的不对称处理——"输家"减记而"赢家"不上调——导致两者差距不断拉大。如今[[内在价值]]远超账面价值，这也是公司愿意在股价低于账面价值120%时[[回购]]股票的原因。

### "五驾马车"与PCC收购

[[BNSF铁路]]承担美国约17%的城际货运量，2015年投入约58亿美元资本支出（近三倍于年折旧费用），税前收益创68亿美元新纪录。"五驾马车"（BNSF、[[伯克希尔哈撒韦能源]]、马蒙、路博润、IMC）合计税前收益达131亿美元。

精密铸件公司(PCC)以320亿美元现金[[收购]]，将成为"六驾马车"新成员。CEO马克·多尼根将PCC打造成世界顶尖航空零部件供应商。[[托德·库姆斯]]在此次收购中发挥了关键作用，是他最初让巴菲特注意到这家公司。

### 保险业务的持续优势

[[保险业]]连续第13年实现承保盈利，2015年贡献18亿美元。[[保险浮存金]]从410亿美元增至880亿美元，13年间承保盈利累计达260亿美元。巴菲特称这是[[伯克希尔哈撒韦]]最大的未入账财富，用48年打造的多层次业务体系无法被复制。

### 四大投资持仓

"四大"投资——美国运通、[[可口可乐]]、[[IBM]]、[[富国银行]]——2015年均有增持。按年末持股计算，应占收益份额达47亿美元，但报告收益仅包含18亿美元股息。巴菲特强调宁愿拥有优秀公司的非控股权益，也不愿拥有平庸公司的100%——"拥有希望之钻的一小部分，也好过拥有整颗水钻"。

### 与3G资本的合作模式

与豪尔赫·保罗·雷曼合作完成亨氏与卡夫合并，[[伯克希尔哈撒韦]]持有[[卡夫亨氏]]约27%股权。两种[[商业模式]]各有特色：3G资本通过快速削减冗余成本提升生产力，而[[伯克希尔哈撒韦]]则从一开始就避免机构臃肿，收购由注重成本的管理者长期经营的企业，采取极端分权方式运营。

### 对美国经济的乐观展望

巴菲特驳斥了美国衰落论，指出美国人均GDP约56,000美元，按实际购买力计算是1930年的六倍。即使2%的实际GDP年增长率，扣除人口增长后人均约1.2%，一代人时间（25年）也能带来惊人收益。"美国的经济魔力依然运转良好。"
