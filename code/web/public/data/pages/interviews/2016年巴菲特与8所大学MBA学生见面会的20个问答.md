# 2016年巴菲特与8所大学MBA学生见面会的20个问答

## 核心要点

- 成功投资者需要控制恐惧和贪婪的能力，高IQ的人也经常恐慌，技术无法取代人类基因里的情绪
- 被动投资（指数基金）长期表现优于主动管理的对冲基金，基金经理越来越富，投资者却遭受损失
- 招聘和投资都应关注"激情"——对事业充满热情的人才能脱颖而出
- 已经富有的人还想翻倍是愚蠢的，不要拿已有的去冒险追求不需要的东西
- 利率之于资产估值，好比引力之于质量，低利率环境下股票相对便宜

## 详细摘要

### 人才与品质

巴菲特认为招聘时最看重的品质是聪明、精力充沛和正直，但最重要的是对所从事业务充满激情。使[[伯克希尔哈撒韦]]70位经理脱颖而出的不是智商，而是对事业的热爱。他建议选择那些你愿意获得其未来收入10%的人作为榜样——有幽默感、值得追随的领导者。

金融行业最重要的技能是销售能力，但能做好的最关键特质是控制恐惧和贪婪的能力。

### 投资策略与市场观点

关于主动与被动管理，巴菲特用100万美元打赌证明标普500指数显著跑赢对冲基金FOF。对冲基金总回报的一半流向管理人，表现却逊于指数40%。他的建议是：找到良好管理的公司，长期持有。

如果今天用100万美元开始，他和[[芒格]]可能只投资四只股票。他从哥伦比亚大学毕业时，将净资产75%投入Geico。投资好公司就无需[[分散投资]]。

关于[[科技与互联网]]，[[托德·库姆斯]]和[[泰德·韦施勒]]各管理90亿美元，投资了苹果。巴菲特承认科技圈在他的能力圈之外，即使微软产品几乎无生产成本且垄断赚钱，他也没有投资。

### 利率与宏观环境

利率对资产估值的影响如同引力对质量。30年国债利率从1982年的14.5%降至2016年的2.5%。如果长期利率保持4%，股票是便宜的。但借钱交易比任何事情更容易使人破产。[[芒格]]说过，聪明人因为"酒、女人和[[杠杆]]"破产。

央行行为对市场影响深远。美联储资产负债表从10年前的1万亿增至4万亿美元，是有史以来最大的对冲基金。没人知道负利率接下来会怎样发展，但"哪里有混乱，哪里就有机会"。

### 保险业务的重要性

1950年巴菲特拜访Geico改变了他的人生轨迹。1986年[[阿吉特·贾恩]]走进他的办公室，30年后管理着[[伯克希尔哈撒韦]]的再保险业务。阿吉特"每个小时都有新创意"，而Geico从1936年后就没改变过。[[保险业]]是伯克希尔的核心业务之一。

### 错误与教训

巴菲特最初的三项业务都已破产：[[伯克希尔哈撒韦]]纺织品业务、Blue Chip Stamps和Baltimore百货商店。他最遗憾的是"那些做了充分调查但没有做、最终错过的事"。关于[[富国银行]]丑闻，他强调出现问题时应"准确定位、迅速获知、然后解决"。
