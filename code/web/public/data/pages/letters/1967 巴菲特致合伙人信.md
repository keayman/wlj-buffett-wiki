# 1967 巴菲特致合伙人信

## 核心要点

- 合伙基金全年上涨35.9%，道指上涨19.0%，超额收益显著，但巴菲特对市场投机风气保持警惕
- 各投资类别表现分化严重：[[套利]]类收益率仅0.89%创历史新低，控制类稳健但相对落后，相对低估类贡献72%的惊人回报
- 通过[[伯克希尔哈撒韦]]完成两笔重要[[收购]]：联合棉布商店和[[国民保险公司]]，对收购结果非常满意
- 重申降低投资目标的决定，强调与优秀的人合作、做有趣的生意比追求更高回报率更重要

## 详细摘要

### 年度业绩与市场观察

1967年合伙基金整体收益19,384,250美元，收益率35.9%，超越道指16.9个百分点。但巴菲特指出，这一年市场中95%以上的股票型基金都跑赢了道指，许多机构收益率超过100%。他引用[[格雷厄姆]]的话警示："投机不缺德、不犯法，也发不了家。"大量资金涌入追求快钱，投机风气盛行，隐藏着风险。

### 投资类别分析

**[[套利]]类**遭遇成立以来最差表现，平均投资17,246,879美元，收益仅153,273美元，收益率0.89%。意外频发是主因，但巴菲特相信长期集中投资的方法仍然有效。

**控制类**平均投资20,192,776美元，收益2,894,571美元。在强劲上涨的市场中相对落后属正常现象。通过多元零售公司和[[伯克希尔哈撒韦]]完成两笔[[收购]]，联合棉布商店和[[国民保险公司]]的表现都超出预期。

**[[低估]]类（相对低估）**表现惊艳，净平均投资19,487,996美元，收益14,096,593美元，收益率72%。一只重仓股（占比达40%上限）连续四年贡献巨大收益，目前已大幅减持。

### 投资理念与目标调整

巴菲特解释了降低投资目标的原因：与一流的人合作、经营感兴趣的生意、获得合理回报（10-12%），比为了多赚几个百分点而东奔西跑更有价值。控制类投资将保持在40%以内，尽管预期收益率可能低于激进的股票买卖。

## 原文金句

> "投机不缺德、不犯法，也发不了家。"

> "要是为了多赚几个百分点，就东一榔头西一棒槌的，太傻了。"

> "本来已经认识了一流的人，和他们合作很愉快，回报率也相当好，就为了回报率再高一些，去惹一肚子气、一身的烦恼，甚至陷入更糟的境地，也不值。"

> "去年，有人天天吃投机的糖，吃成了大胖子，赚了很多钱。虽然我们还是吃燕麦，但是什么时候整体股市都患上消化不良，别以为我们能不疼不痒。"
