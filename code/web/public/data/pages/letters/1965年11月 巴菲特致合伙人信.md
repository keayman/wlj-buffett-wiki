# 1965年11月 巴菲特致合伙人信

## 核心要点

- 合伙基金已取得[[伯克希尔哈撒韦]]控股权益，对控股公司的估值采用资产价值而非市场价格
- 新增第七条基本原则：单笔投资最高可占净资产40%，体现集中投资理念
- 业绩评价应以道指为基准，且至少看三年以上周期
- 投资决策基于价值而非流行，追求极大的[[安全边际]]

## 详细摘要

### 年度事务安排

信中详细说明了1966年的承诺书流程：承诺书可在12月31日前修改，之后不可更改。提现将于1月5日支付，追加投资应在1月10日前存入。11月存入的追加资金可获6%利息，12月存入则无利息。

### [[伯克希尔哈撒韦]]控股权益估值

合伙基金已取得[[伯克希尔哈撒韦]]的控股权益。巴菲特强调，对于控股公司，资产价值和盈利能力是估值的主导因素，市场价格仅对少数股东权益有意义。年末估值将介于净流动资产价值和账面价值之间，固定资产按50%折价计算。这种方法对追加投资和提现的合伙人都公平。

### 基本原则要点

巴菲特重申了合伙基金的核心原则：不做收益率保证；业绩以市值变化衡量而非应税所得；评价标准是与道指和大型基金对比；评价周期至少三年。他承诺：选择投资依据价值而非流行；追求极大[[安全边际]]并[[分散投资]]以降低永久性资本损失风险；全家所有净资产都投在合伙基金里。

### 集中投资的新原则

新增的第七条原则允许单笔投资最高占净资产40%，前提是掌握充分的事实和逻辑、正确概率极大、且[[内在价值]]不会发生根本变化。这一修改源于合伙基金用40%净资产投资了美国运通，体现了巴菲特"与大多数同行相比，分散程度远远更低"的投资风格。

## 原文金句

> "我们选择投资的依据是价值高低，不是流行与否。"

> "我们在每一笔投资中都追求极大的安全边际并分散投资，力图将永久性资本损失（不是短期账面亏损）的风险降到绝对最小值。"

> "与大多数同行相比，我们的分散程度远远更低。"

> "评价表现应该看五年，至少要看三年，低于三年的业绩没意义。"
