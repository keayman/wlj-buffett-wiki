# 1998年巴菲特与比尔盖茨在华盛顿大学的对话

## 核心要点

- 成功与智商无关，关键在于理智——将智商比作汽车马力，输出功率取决于理智程度
- 成功的定义是快乐，与欣赏的人共事、做喜欢的事情是安身立命之本
- 投资不需要很多交易就能成功，关键是找到可预测、不变的东西
- 习惯的力量巨大，年轻时培养好习惯将决定一生的成就
- 技术革命虽然改变世界，但投资者应关注不被技术改变的企业

## 详细摘要

### 成功的本质：理智胜于智商

[[巴菲特]]认为成功与智商无关，关键在于理智。他将智商比作汽车马力，而输出功率取决于理智程度。200马力的车百分之百发挥，胜过400马力只发挥100马力。许多聪明人做出妨碍自己的事情，原因在于习惯、性格和禀性。

他建议选择最崇拜的人，写出其优点并加以模仿；同时找出最厌恶的人，避免其缺点。习惯的链条起初太轻以至于察觉不到，直到有一天太重而无法截断。

### 快乐是成功的定义

[[巴菲特]]将成功定义为快乐。他庆幸每天都做着喜欢的事情，和欣赏的人一起工作。他跳着踢踏舞去上班，到了办公室如同躺在沙发上绘制天花板。

他曾拒绝一些本可接受的交易，原因是不喜欢与某些人共事。纯粹为了钱而结婚是糟糕的主意，如果已经非常富有还这样做，那绝对是神经有问题。

盖茨同意这一观点，认为关键在于从每天工作中得到乐趣，与有魅力的人共事，致力于解决新问题。

### 投资决策与合作伙伴

[[巴菲特]]认为最好的经营决策是参与合伙经营。投资不需要很多交易就能成功，如果有20个孔的穿孔卡片，做出20项好决定就能变得非常富裕。

他承认买下[[伯克希尔哈撒韦]]可能是最不理想的决定，这是"捡烟蒂"投资法的典型案例。他不遗憾没有收购微软，因为不了解这家企业，但对于了解的企业如房利美，没有执行投资决定让他错失了数十上百亿美元。

[[芒格]]是[[巴菲特]]多年的合作伙伴。[[芒格]]经常说[[巴菲特]]做的事情很愚蠢，如果他说"确实很愚蠢"，那是真的；如果只说"很愚蠢"，就当他投了赞成票。

### 技术革命与投资策略

[[巴菲特]]坦言一开始没有领悟到技术革命的巨大影响。他对待技术革命的态度与盖茨截然相反——在技术革命中寻找商机时，他关注的中心是不变的东西。

他看待因特网的方式是确定一个行业或公司会受到怎样的损害，然后尽力避免。以箭牌口香糖为例，因特网不会改变人们咀嚼口香糖的方式，也不会改变[[可口可乐]]作为人们偏爱饮料的地位，或者人们刮胡子的习惯。

他寻求可预测的东西，在盖茨所做的事情中发现不了这种可预测性。作为社会成员他赞同技术变革，但作为投资者始终以警惕的眼光看待。

### 高科技与商业效率

[[巴菲特]]坚信高科技在企业效率方面起到很大推动作用，但提出一个深刻的比喻：商业上的很多事情就如同盛大的乐队沿街演出，如果想观看就不得不踮起脚尖，30秒后其他人也都踮起脚尖，双腿累得抽筋却还是很难看到盛况。

当创新取得进步时，资本主义似乎自行失效了。真正的难题在于如何在踮起脚尖的同时又不让别人注意到你。

### 继任者与公司未来

[[巴菲特]]表示即使今夜死去，人们不会喝不到[[可口可乐]]，不会放弃刮胡子，不会少吃[[喜诗糖果]]。这些公司拥有非常好的产品和杰出的管理人员。他已经确定了继任者人选，董事会也知道他们是谁，并准备了一封以"昨天我死了"开头的信说明公司计划。

他指出[[伯克希尔哈撒韦]]和微软的差别在于电脑业的特殊性——在瞬息万变的旋涡中，很难保证五年后某个人对公司经营、技术业务及全球环境的适应程度。

### 财富回馈社会

[[巴菲特]]提出"子宫乐透"思想实验：假设出生前24小时可以制定社会法则，但不知道自己将出生在什么家庭。你想要的系统应该能带来更多商品和服务，让最有才能的人每天工作超过12小时，同时照顾到那些不太完美、缺乏运气的人，让人们不再恐惧。

他表示自己财产的99%以上将回归社会，因为社会对待他们太好了。盖茨同意这一哲学，认为把大量金钱传给人对接受者可能是件坏事。
