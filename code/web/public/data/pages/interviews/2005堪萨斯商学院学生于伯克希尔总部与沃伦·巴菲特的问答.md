# 2005堪萨斯商学院学生于伯克希尔总部与沃伦·巴菲特的问答

## 核心要点

- 投资是没有终点的旅程，真正的成功不以财富多少衡量，而在于享受过程本身
- 市场并非永远有效，"股价大多数时候合理"与"股价永远合理"有天壤之别，这个差别造就了财富的分化
- 投资第一步是把公司分成两堆：能看懂的和看不懂的，把"太难"的放一边，专注于能力圈内的机会
- 选择益友、选对伴侣是人生最重要的决定，年轻时应专注于投资自己

## 详细摘要

### 投资的动力与人生态度

巴菲特表示自己不以赚钱多少来衡量成功，财产多几位数并不能带来更多幸福。投资对他而言是一个没有终点的旅程，总有新问题、新乐趣，而且新问题与旧问题相似，可以运用过去积累的知识。他打算一直做下去，除非老年痴呆——但他幽默地说，得了老年痴呆自己不知道，所以要求三个孩子必须一起告诉他。

### 友谊与合作伙伴的价值

巴菲特强调交朋友要选择益友。他与比尔·盖茨一家旅行过四次，盖茨为他考虑得特别周到，甚至在中国之行时派厨师提前教中国酒店制作汉堡。他的另一位好朋友[[芒格]]帮助他转向购买好生意，不再像从前那样"捡垃圾"。巴菲特说，他更欣赏的是[[芒格]]的为人处世，多年合作中双方都受益颇多。

### 对[[格雷厄姆]]的评价

[[格雷厄姆]]不把财富积累作为追求目标，投资只是他的副业。他天生有学者气质，希望研究出一套普通人都能用的量化方法。巴菲特总结[[格雷厄姆]]的智慧为三个要点：买股票就是买公司；股市短期是投票器、长期是称重机；[[安全边际]]。

### 对[[有效市场]]理论的反驳

巴菲特认为市场是相当有效的，大多数股票价格合理，但"股价大多数时候合理"和"股价永远合理"有天壤之别。他以[[华盛顿邮报]]为例：当年股市报价1亿美元，但拍卖的话至少有人出价4亿美元。他展示了1951年的《穆迪手册》，其中西部保险证券每股盈利21-29美元，股价却只有3-13美元。他还展示了2004年韩国股票手册，用五六个小时就找到了20-25只低估股票，构建了1亿美元的投资组合。

### 寻找投资机会的方法

第一步是把公司分成两堆：能看懂的和看不懂的。把搞不懂或难以预测的放到"太难"那一堆。然后大量阅读年报、季报，了解行业和竞争对手。巴菲特强调分析之前不看股价，先估算价值再与市场报价比较，避免价格影响判断。投资就像在股市中寻找赛马术语中的"overlay"，要不断扩大自己的能力圈。

### 关于现金配置与等待机会

[[伯克希尔哈撒韦]]持有400亿美元现金，巴菲特坦言找不到机会虽然难受，但乱投资要遭殃。他回忆1998年长期资本管理公司崩盘时出现了大机会，但他当时在黄石公园度假，错过了可能赚20亿美元的机会。关键是要时刻准备着行动。

### 海外投资经验

巴菲特谈到投资[[中国石油]]的经历：三年前读了一份报告，发现其估值只有其他石油巨头的三分之一。虽然希望资产在美国，但不想花3倍价钱买。公司承诺每年将利润40%用于分红，投资4亿美元后每年收到1亿美元股息。但他表示投资中国的银行和[[保险业]]公司不太放心，因为看不懂。

### 对年轻人的建议

巴菲特建议做自己喜欢的事，别担心钱，别做无法全身心投入的工作，别和不喜欢、不信任的人共事。他强调人生最重要的决定是选择伴侣，最重要的工作是养儿育女。年轻时正是投资自己的年龄，努力提升自己比拥有1亿美元更重要。

## 原文金句

> "我不以赚钱多少或复合收益率高低来衡量成功。财产多几位数，不能过得更幸福。我做的事没有终点。这是一个没有终点的旅程。"

> "市场是相当有效的，大多数股票的价格是合理的。'股价大多数时候合理'，'股价永远合理'，这两种说法有天壤之别。因为这个天壤之别，有人发家致富，有人长期贫穷。"

> "找不到机会虽然难受，但乱投资要遭殃。"

> "现在，你们是大学生，正是投资自己的年龄，现在投资，将来才能获益。你们自己就是自己的资产，努力提升自己吧。"

> "我喜欢始终能生钱的资产，黄金不会下蛋。"
