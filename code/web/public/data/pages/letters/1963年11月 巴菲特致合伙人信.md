# 1963年11月 巴菲特致合伙人信

## 核心要点

- 强调"基本原则"的重要性：合伙人必须彻底理解并真心认同投资理念，否则不应签署协议
- 登普斯特农具公司成功出售，验证了"买得便宜就已锁定利润"的投资逻辑
- 截至10月末，合伙基金收益率约32%，大幅领先道指18.8%的涨幅
- 保持理性预期：13.2个百分点的领先优势属于异常年份，长期稳定小幅领先才是合理目标

## 详细摘要

### 年度行政事务

巴菲特随信附上1964年合伙协议、承诺书及"基本原则"文件。他特别强调，每位合伙人应每年至少阅读一遍"基本原则"，并要求做到"不只是脑袋里明白，还要融入骨髓"。若无法真心认同这些理念，就不应签署协议。

提款和追加投资均无金额限制，11月存入的追加资金按6%计息，12月存入则不享有利息。

### 投资业绩表现

截至10月末，道指涨幅18.8%，合伙基金整体收益率约32%。扣除普通合伙人分成后，有限合伙人收益率约25.5%。

三个投资类别——[[低估]]类、[[套利]]类、控股类——今年表现都很出色。约300万美元的总收益中，200余万来自有价证券，近100万来自出售登普斯特经营资产。

### 登普斯特案例的启示

登普斯特农具公司以持续经营企业的标准顺利出售。巴菲特指出，这笔投资"从进场那一刻起就已锁定利润"，体现了[[买入价格]]的决定性作用。哈里·博特尔的出色管理工作为这笔投资"额外添了好几道甜点"，他将在1964年成为有限合伙人。

### 理性的业绩预期

巴菲特坦率地表示，如果全年相对道指的领先优势维持在13.2个百分点，他会将其视为"异常年份"。他强调："长期而言，我不认为相对道指领先13.2个百分点是可以持续的。每年能比较稳定地小幅领先道指，我就已经相当满足了。"
