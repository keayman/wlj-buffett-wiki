# 2024 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]2024年向美国国税局缴纳268亿美元企业所得税，创下美国企业纪录，约占全美企业所得税总额的5%
- 经营收益达474亿美元，[[保险业]]尤其是盖可保险表现出色，[[托德·库姆斯]]五年改革成效显著
- 60年来仅发放过一次股息，持续再投资策略使公司不断壮大，体现[[复利]]的长期魔力
- 承认错误是CEO的责任，拖延纠正错误是最不可饶恕的罪过
- 绝大部分资金将永远投入股权资产，主要是美国股票，现金永远不会比优质企业更重要

## 详细摘要

### 关于坦诚报告与承认错误

[[伯克希尔哈撒韦]]的"报告"意味着超越法律要求的责任——向股东如实汇报好消息和坏消息。在2019至2023年间，巴菲特在信中使用了16次"错误"或"失误"，而许多大公司从未用过这类词。

[[芒格]]曾告诫：问题不会因一厢情愿而消失，必须采取行动。最不可饶恕的罪过是拖延纠正错误，即"吮手指"。[[格雷格·阿贝尔]]认同这一信条，深知一旦开始糊弄股东，很快就会连自己也信了那套鬼话。

### 皮特·利格尔的传奇

森林河公司创始人皮特·利格尔为股东贡献了数十亿美元财富。2005年的[[收购]]过程干脆利落：一封信介绍数据和期望售价，一周内见面，几分钟内达成协议。皮特主动要求年薪10万美元——因为不想挣得比老板多——外加超额利润的10%作为奖金。

这个故事说明：商业才能很大程度上是与生俱来的，先天远胜后天。选择CEO从不看候选人上过什么学校。

### 2024年业绩表现

经营收益474亿美元，超出预期。[[保险业]]承保利润从54亿美元增至90亿美元，投资收入从96亿美元增至137亿美元。[[BNSF铁路]]和[[伯克希尔哈撒韦能源]]合计盈利有所改善，年末将公用事业持股比例提高到100%。

巴菲特强调经营收益指标而非GAAP收益，因为股票和债券的资本利得或损失会剧烈波动且无法预测，而持有期限往往是几十年。

### 破纪录的税款与再投资

60年前接管时，[[伯克希尔哈撒韦]]已连续十年未缴一分钱所得税。如今累计缴税超过1010亿美元。2024年的268亿美元税款意味着：即使每20分钟寄出一张100万美元支票，全年366天也不够。

关键因素是60年来仅发放过一次股息（1967年共101,755美元），股东支持持续将[[留存收益]]再投资，体现了持续储蓄文化与长期[[复利]]魔力的完美结合。

### 投资哲学与资本配置

股权投资双管齐下：189家控股子公司（通常100%持股）和十几家上市公司的少数股权（如苹果、美国运通、[[可口可乐]]、穆迪），年底市值2720亿美元。

[[伯克希尔哈撒韦]]永远不会把持有现金等价物看得比拥有优质企业更重要。纸币价值可能因财政胡来而灰飞烟灭，但企业通常能找到应对货币不稳定的办法。

### 财产意外险的独特模式

[[保险业]]的财务模式在大型企业中罕见：先收保费，很久之后才知道成本——有时延后30年甚至更久。这种模式的好处是在承担费用前就拥有现金（[[保险浮存金]]），但风险是可能在意识到之前已经巨亏。
