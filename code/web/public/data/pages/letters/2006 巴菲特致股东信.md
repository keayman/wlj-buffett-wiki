# 2006 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]净资产增加169亿美元，每股账面价值增长18.4%，42年复合增长率达21.4%
- [[保险浮存金]]从1967年的1,600万美元增长至509亿美元，且37年中有35年实现[[承保纪律]]盈利
- 通用再保险终于走出困境，衍生品合约从23,218份减至197份，累计税前亏损4.09亿美元的噩梦结束
- [[盖可保险]]市场份额从5.6%升至6.1%，在获客效率上远超竞争对手
- 住宅房地产市场因放贷标准松弛开始出现问题，"明天的事明天再说"的心态正在反噬借款人

## 详细摘要

### 保险业务：伯克希尔的核心引擎

[[保险业]]是[[伯克希尔哈撒韦]]的核心业务。[[保险浮存金]]——虽不属于公司但可用于投资的资金——是理解保险经济特性的关键。当[[承保纪律]]持续产生利润时，浮存金的价值甚至超过同等规模的净资产，相当于使用免费资金赚钱。

大多数保险公司入不敷出，但伯克希尔37年中有35年实现承保盈利。严格的风险评估纪律贯穿每一个决策，虽无法消除所有失误，但确保远离灾难性错误。

### 通用再保险：浪子回头

1998年[[收购]]通用再保险时，巴菲特犯了两个错误：没做尽职调查就买下，以及用股票而非现金支付。[[芒格]]称这种仓促行事为"吮手指"。

乔·布兰登和塔德·蒙特罗斯从2001年起整顿承保业务，顶住了行业抢份额的压力。2006年通用再保险终于跻身全球顶尖再保险公司之列。衍生品账本从23,218份合约减至197份，累计税前亏损4.09亿美元——这头疯狂的大象终于被驯服。

### 盖可保险：低成本模式的胜利

[[盖可保险]]保单量增长8.1%，市场份额升至约6.1%，营收达119亿美元。广告投入约6.31亿美元，壁虎和穴居人广告效果显著。托尼·奈斯利掌管十余年，同时做到了扩大份额和维持承保利润这两件看似矛盾的事。

### 阿吉特·贾恩：无价之宝

[[阿吉特·贾恩]]1986年加入伯克希尔，此后创造的利润和浮存金堪称天文数字。他对概率论有直觉般的理解，能评估陌生复杂的风险，能在别人说"好"时说"不"，还能在灾难发生后迅速赔付。他的存在使伯克希尔成为全球最有价值的再保险集团。

### 利捷航空：扭亏为盈

[[利捷航空]]2006年终于实现全面盈利。欧洲业务曾是大坑，5年只获得80个客户，累计税前损失2.12亿美元。但2005-2006年新增589名客户，在马克·布斯领导下扭亏为盈。全球税前收益达1.43亿美元。利捷航空管理的机队价值远超前三大竞争对手总和。

### 中美能源：稳健的公用事业

[[伯克希尔哈撒韦能源]]（原[[中美能源]]）持有多种公用事业资产，包括英国第三大电力分销商、爱荷华电力公司、西部六州电力公司，以及输送全美约8%天然气的管道公司。合伙人包括沃尔特·斯科特、戴夫·索科尔和[[格雷格·阿贝尔]]。

旗下房地产经纪公司美国家庭服务因住宅市场放缓，成交额下降9%，利润缩水50%。放贷标准松弛导致的问题开始显现——那些"可选择性"合同和"诱惑性"利率让借款人面临高于正常的月供。

### 克莱顿预制房：行业异类

[[克莱顿房屋]]在行业创下1962年以来最低销量时，仍赚取5.13亿美元税前利润。凯文·克莱顿对这门生意了如指掌，是理性的决策者。
