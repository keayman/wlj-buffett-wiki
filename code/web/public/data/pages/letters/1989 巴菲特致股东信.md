# 1989 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]净资产增长44.4%，25年复合增长率达23.8%，但[[内在价值]]增长率实际更高
- 投资组合的"估值修复"红利已基本兑现，未来只能依靠企业自身价值增长这"单重收获"
- 高增长率终将自我终结——基数越大，维持高增长越困难
- 长期持有策略具有显著的[[税收效率]]优势，但更重要的是珍惜美好的商业关系
- "七加一圣徒"非保险业务表现卓越，税后收益达平均股东权益的57%

## 详细摘要

### 内在价值与账面价值的关系

[[伯克希尔哈撒韦]]1989年净资产增加15.15亿美元。真正重要的是[[内在价值]]——通过对企业未来所有现金流折现计算得出。25年前账面价值19.46美元时，[[内在价值]]更低，因为资产绑在纺织业务上；如今大部分业务价值远超账面价值，这种从折价到溢价的演变意味着[[内在价值]]复合增长率比账面价值23.8%的增长率更高。

### 未来增长的双重挑战

投资组合中的优秀企业已获市场广泛认可，过去被不合理[[低估]]的情况已不复存在。过去的成绩得益于"双重收获"：企业[[内在价值]]增长加上市场估值修复。未来只能靠"单重收获"了。

在有限的世界里，高增长率终将自我终结。[[芒格]]和巴菲特虽然没找到每15分钟翻一番的方法，但以49亿美元的基数实现15%年增长，难度远超从2,200万美元起步时的23.8%。

### 税务与长期投资的数学优势

1989年缴纳约7.12亿美元所得税，其中1.72亿美元当期支付，5.4亿美元递延处理。递延税款相当于财政部的无息贷款，到期日由[[伯克希尔哈撒韦]]自己决定。

通过极端对比说明[[税收效率]]：频繁交易20年后剩约25,250美元，而持有一笔20年翻20倍的投资最终到手约692,000美元——差距惊人，唯一原因是纳税时间不同。

但长期投资策略的真正原因并非税务优势，而是美好的商业关系稀有且令人愉悦。放弃与令人钦佩的人相处，去和平庸之辈打交道，就像已经富有还为钱结婚——简直是疯了。

### 透视收益法

衡量[[伯克希尔哈撒韦]]基本盈利能力的最佳方法是"透视收益"法：报告经营收益加上被投资公司[[留存收益]]中属于我们的份额。五大主要被投资公司支付股利约4,500万美元，但[[留存收益]]应占份额达2.12亿美元。这些[[留存收益]]由才华横溢、以股东利益为导向的管理者运用，价值等同甚至更高。

### 非保险业务的卓越表现

"七加一圣徒"——[[波仙珠宝]]、[[布法罗新闻报]]、菲希海默兄弟、柯比吸尘器、[[内布拉斯加家具店]]、斯科特费泽制造集团、[[喜诗糖果]]和[[世界图书百科全书]]——汇集了经济特质从优秀到卓越的企业。税后收益达平均股东权益的57%，且在零净[[杠杆]]条件下实现。

[[波仙珠宝]]加入首年销售额大幅增长，费用率约为同类珠宝店三分之一。[[喜诗糖果]]销售磅数增长8%，广告预算从400万增至500万美元，报纸广告回报极其惊人。[[内布拉斯加家具店]]创下销售额和利润新纪录，[[B夫人]]虽因意见分歧辞职，但96岁高龄又开始了新生意。

## 原文金句

> "真正重要的是内在价值——这个数字代表我们旗下所有业务合理价值的总和。"

> "在一个有限的世界里，高增长率终将自我终结。如果增长的基数很小，这条定律可能暂时不起作用。但当基数膨胀到一定程度，好日子就到头了：高增长率最终会铸造自己的锚链。"

> "我们发现美好的商业关系如此稀有、如此令人愉悦，以至于我们想珍惜已经建立起来的每一段关系。"

> "放弃与那些我们深知有趣且令人钦佩的人相处的时间，去和那些我们不了解、且人格特质很可能接近平庸水准的人打交道，实在是毫无道理。这就好比为了钱而结婚——在大多数情况下是错误的，如果本身已经很富有了，那简直就是疯了。"
