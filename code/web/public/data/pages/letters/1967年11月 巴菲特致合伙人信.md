# 1967年11月 巴菲特致合伙人信

## 核心要点

- 合伙人需在12月31日前确认承诺书，逾期不可更改
- 今年卖出大量仓位，实现2400万美元收益，合伙人需关注纳税影响
- 当前约3600万美元资金（2000万控股公司+1600万短期国债）未参与股市，因找不到安全且明显能赚钱的机会
- 修改了"基本原则"第四、五条，调整了业绩目标，建议不满意者年末提现

## 详细摘要

### 年度事务安排

巴菲特随信附上1968年承诺书、税收数据信函及相关副本。承诺书需在12月31日前确认，届时将转交毕马威会计师事务所，不可更改。今年实现了2400万美元的投资收益，合伙人应仔细了解纳税情况。

提现将于1月5日支付，金额无限制；追加投资应在1月10日前存入，11月存入可获6%利息。

### 业绩表现

截至写信时，道指约880点，年内上涨约15%。合伙人的业绩上涨约27%，有限合伙人收益率约22%。控股权益估值暂未调整，仍采用1966年年末数据，[[伯克希尔哈撒韦]]和多元零售的新估值将在12月22日信中公布。

### 投资策略与现金配置

巴菲特坦言当前约3600万美元未参与股市——2000万投资于控股公司，1600万买入短期国债。这并非基于[[市场预测]]，而是从长期[[内在价值]]角度找不到安全且明显能赚钱的机会。他提醒合伙人：若市场短期大涨，不要对业绩有过高期望。

### 基本原则调整

因环境变化，"基本原则"第四、五条已完全修改。巴菲特明确表示，若合伙人对新目标不满意，建议年末提现，将资金投入收益更高的地方。

## 原文金句

> "这不是因为我对市场走向有什么判断，只是我找不到任何很明显能赚钱、又安全的机会（从长期价值的角度考虑，不是短期的市场报价），没地方投资这些钱。"

> "往年，新的一年开始时，我们总是有一些就要孵出小鸡的蛋，今年，到现在鸡窝里还空空如也。"

> "如果这个目标，就算实现了，你也不满意，你最好在年末把自己的资金提出去。"
