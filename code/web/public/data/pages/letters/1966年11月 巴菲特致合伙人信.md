# 1966年11月 巴菲特致合伙人信

## 核心要点

- 合伙基金今年业绩高于平均水平，领先道指约17个百分点，但下半年比上半年困难
- 详细说明了控股公司与少数股权的不同估值方法，强调资产因素在控股权益估值中的重要性
- 禁止合伙人代替他人间接投资，以维护合伙基金的纳税地位和避免潜在纠纷
- 市场波动可能导致年度净亏损，若收益率低于6%，合伙人账户资金将减少

## 详细摘要

### 年度业绩表现

今年业绩高于平均水平，但下半年明显比上半年艰难。六月份之前，[[低估]]类投资大幅领先道指，此后略微落后。[[套利]]类投资规模虽小但收益不错，基本弥补了下半年的差距。按道指810点计算，道指下跌14%，而合伙基金上涨4%（不计利息）。市场波动剧烈，若道指继续大跌，基金也必然受影响。

### 控股权益估值方法

对于少数股东持仓，完全使用市场价格估值。但当处于控股地位时，拥有的是公司所有权而非股票，必须采用不同方法。控股权益价格可能是少数权益的60%到500%，这证明了差异化估值的必要性。

估值主要考虑盈利能力和资产价值。对[[伯克希尔哈撒韦]]的估值，选取介于净流动资产价值和账面价值之间的数值。今年年末估值很可能远高于少数权益市场价值。对霍赫希尔德·科恩公司，则用[[买入价格]]加上买入后分得的[[留存收益]]计算。

### 估值的动态调整

市场价值波动有时会影响企业估值，但对资产因素占比重的企业影响较小。若证券市场和货币市场情况有现实必要，可能在12月末对伯克希尔和科恩公司进行减值，届时会提前通知合伙人。

### 合伙人投资规则

合伙基金已停止接纳新合伙人（除非贡献不只是资金）。《合伙协议》第9条禁止合伙人代替亲友间接投资，这关系到1961年财政部裁决书确定的合伙纳税地位。私下安排可能因身故、离婚、税务检查等情况引发麻烦。

## 原文金句

> "当我们持股超过50%或者不到50%但处于控股地位时，我们就拥有一家公司的所有权，而不是股票，这时必须使用不同的估值方法。"

> "就在同一时间，控股权益的价格可能是少数权益的60%到500%，这足以证明对控股权益要采用不同的估值方法。"

> "我有责任对控股权益进行公允估值，这个公允是要对特定时间追加投资和提取投资的合伙人都公允。"
