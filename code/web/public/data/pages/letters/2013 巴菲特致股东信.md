# 2013 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]净资产增加342亿美元，每股账面价值增长18.2%，49年复合增长率达19.7%
- 完成两项重大收购：内华达能源公司（56亿美元）和亨氏公司（约180亿美元），后者创建了与3G资本合作的新模板
- "五驾马车"（中美能源、BNSF、伊斯卡、路博润、马蒙）税前利润达创纪录的108亿美元
- 保险业务连续第11年实现承保盈利，[[保险浮存金]]从410亿增至770亿美元
- "四巨头"投资（美国运通、[[可口可乐]]、[[IBM]]、[[富国银行]]）持股比例全部提升

## 详细摘要

### 业绩表现与价值衡量

2013年[[伯克希尔哈撒韦]]每股账面价值增长18.2%，但在标普500指数大涨32.4%的强劲市场中暂时落后。[[芒格]]和巴菲特认为这符合预期：在市场温和或下跌年份能跑赢，强劲上涨年份会暂时落后。从2007年底到2013年底的完整周期看，伯克希尔跑赢了标普。

公司[[内在价值]]远超账面价值，且差距近年大幅扩大。这解释了为何以账面价值120%回购股票对继续持有的股东有利。

### 重大收购与合作模式

亨氏收购创建了新的合作模板：[[伯克希尔哈撒韦]]购买80亿美元优先股（年回报约12%），与3G资本各出资42.5亿美元购买普通股。关键区别在于伯克希尔从不打算出售，这与私募股权的退出导向截然不同。

内华达能源公司被[[伯克希尔哈撒韦能源]]（原[[中美能源]]）以56亿美元收购，为可再生能源投资提供了大量机会。

### 保险业务的核心地位

[[保险业]]是伯克希尔的核心引擎。财产意外险的"先收保费、后理赔"模式创造了[[保险浮存金]]——这些资金虽不属于公司，但可用于投资。自1967年以860万美元收购[[国民保险公司]]以来，浮存金从3900万增至772亿美元，11年间累计实现220亿美元税前承保利润。

盖可保险已从1996年的第七大车险公司升至第二，秘诀简单：低价和可靠服务。

### 投资组合管理

"四巨头"2013年利润中伯克希尔份额约44亿美元，但报表只反映14亿股息。那30亿留存收益同样有价值——用于[[回购]]和再投资，提升伯克希尔未来收益份额。四巨头未实现收益累计达390亿美元。

[[托德·库姆斯]]和[[泰德·韦施勒]]各管理超70亿美元组合，业绩轻松跑赢标普，也超过了巴菲特本人。

### 对美国的信心

收购[[BNSF铁路]]被称为"对美国经济前途的全押"。过去237年，谁曾因看空美国而获益？[[市场先生]]的内在活力将继续施展魔法，美国最好的日子还在前头。
