# 2021 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]拥有全美最多的基础设施资产（1,580亿美元），并每天向美国财政部缴纳约900万美元税款
- [[保险浮存金]]从1967年的1,900万美元增长至1,470亿美元，使用成本低于零
- "四巨头"——保险集群、苹果、[[BNSF铁路]]、[[伯克希尔哈撒韦能源]]——贡献了公司大部分价值
- 过去两年回购了9%的流通股，花费517亿美元，这是当前最确定的价值创造方式
- 持有1,440亿美元现金并非失去投资热情，而是找不到符合长期持有标准的标的

## 详细摘要

### 伯克希尔的真实面貌

很多人把[[伯克希尔哈撒韦]]看作金融资产集合体，但事实上公司拥有的美国基础设施资产比任何其他美国公司都多。公司与美国政府之间存在一种"看不见的财务合伙关系"——1955年合并时每天只缴100美元税，如今每天缴纳约900万美元。[[复利]]的力量加上收益再投资，让股东和财政部都成为受益者。

### 保险帝国的价值

自1967年以860万美元收购[[国民保险公司]]以来，[[保险浮存金]]已增长至1,470亿美元。这笔资金使用成本低于零，且具有极强的粘性，让公司可以着眼长远进行投资。[[阿吉特·贾恩]]1986年加入时毫无保险经验，但35年后证明他是最完美的人选。[[伯克希尔哈撒韦]]在应对灾难性事件方面的实力无人能及。

### 四巨头的贡献

保险集群排在首位，产品永不过时，保费随经济增长而增加。苹果公司持股5.55%，[[回购]]让伯克希尔不花一分钱就增加了持股比例，"应得"收益高达56亿美元。[[BNSF铁路]]创下60亿美元利润纪录，是美国商业第一大动脉。[[伯克希尔哈撒韦能源]]利润从2000年的1.22亿美元增至40亿美元，在风能、太阳能和输电方面成为行业领军者，[[格雷格·阿贝尔]]功不可没。

### 现金与回购策略

1,440亿美元现金中1,200亿投资于美国国库券。公司承诺永远持有超过300亿美元现金，确保财务安全。当[[收购]]和股票投资都缺乏吸引力时，[[回购]]成为最确定的价值创造方式。长期低利率推升了所有生产性投资的[[买入价格]]，让人难以找到令人兴奋的标的。
