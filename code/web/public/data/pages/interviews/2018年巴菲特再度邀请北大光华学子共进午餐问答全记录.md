# 2018年巴菲特再度邀请北大光华学子共进午餐问答全记录

## 核心要点

- 最好的生意是拥有宽广[[护城河]]的企业，但护城河会随时代变化而消失，如报业被互联网侵蚀
- 投资决策应由业务本身的性质决定，而非宏观经济或利率因素，不要让政治信仰影响投资信念
- 寻找被明显错误定价的机会，找到1-2个就足以致富；找不到机会时，持有现金是明智选择
- 做自己有热情且擅长的事，人们自然会来找你；个人生活的幸福会帮助职业发展
- 美国成功的秘密武器是释放人才潜能的市场体系，中国近年的巨变也证明了这一点

## 详细摘要

### 报业兴衰与护城河的演变

[[巴菲特]]回顾了报业从1880年代的繁荣到如今衰落的历程。报纸曾是具有地方性自然垄断力量的行业，但[[科技与互联网]]的出现彻底改变了这一格局。电视台和互联网取代报纸成为主要信息来源，报业逐渐失去了用户群和垄断力量。

谈及[[华盛顿邮报]]的投资，当时整个公司售价8000万美元，但资产价值达4-5亿美元，这是不需要多加考量就可以做的决策，最终获得超过100倍回报。

最好的产业是有[[护城河]]的城堡。Google是优秀企业的典范，其护城河大到竞争者Bing被戏称为"But It's Not Google"。

### 投资决策的核心原则

[[伯克希尔哈撒韦]]从不根据宏观经济考虑做决策。投资决策应由业务本身的性质和特点决定，而不是由利率决定。[[巴菲特]]强调不会让政治信仰影响投资信念。

在评估公司时，需要关注的因素包括：[[留存收益]]、[[品牌]]与商标的力量、管理层对公司的热情、盈利能力与现金流。最值得投资的公司是那些已经好到无须考虑管理层的公司。

投资如同棒球击球，但与棒球不同的是，投资者可以选择"不挥棒"。没有明确的"好球带"，除非你足够了解公司及其投资潜能。

### 寻找被低估的机会

真正的诀窍是找到那些向你大喊的明显机会。从成千上万的公司里找到真正有潜力的公司并正确评估它们，只需找到1-2个被明显[[低估]]的机会就可以变得非常有钱。如果找不到任何东西，可以选择持有现金。

很多人认为波动性是风险，但实际上它对投资者有好处，因为真正想要的是了解的资产的市场价格与其[[内在价值]]之间存在巨大差异。

### 人工智能与投资

平均而言，机器在投资决策中可能超越人类，但最好的人类会击败机器。AI可能比标准普尔500做得更好，但没有一台机器可以击败[[芒格]]。

让机器决定投资和资金分配是非常危险的。自动化订单是1997年和2008年崩盘的原因之一。拥有正确投资理念的人将不受机器影响，继续做好投资。

### 美国经济成功的秘密

美国的秘密武器是一个释放人才潜能的体系。当年美国能比中国进步更快的原因是，美国在释放人的潜能。如今中国已发生翻天覆地的变化，显示出巨大的社会潜力。通过分工越来越专业化的市场体系来释放人的潜能，这是[[巴菲特]]非常乐观的原因。

### 热情与职业选择

如果今天重新开始职业生涯，[[巴菲特]]仍然只会做有热情的工作。他每天都期待着工作，是跳着踢踏舞去上班的。如果去做有激情且擅长的事，人们就会来找你。

个人生活的幸福会帮助职业发展。与相关联的人保持友好关系很重要，比如与[[芒格]]自1959年起从未有过一次争吵。人生最重要的搭档是配偶。
