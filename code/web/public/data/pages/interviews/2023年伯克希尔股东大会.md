# 2023年伯克希尔股东大会

## 核心要点

- [[伯克希尔哈撒韦]]一季度经营利润约80亿美元，预计全年经营利润将高于去年，主要得益于投资收益增加和保险业务改善
- 大多数子公司今年盈利可能不如去年，经济热度正在消退，从"供不应求"转向需要促销
- [[伯克希尔哈撒韦]]净资产达5040亿美元，是美国净资产最大的公司；[[保险浮存金]]达1650亿美元，是宝贵的零成本资金来源
- 盖可保险在科技方面欠账太多，正在努力补足短板，目标综合成本率96%

## 详细摘要

### 一季度业绩与经济展望

[[巴菲特]]开场介绍了[[伯克希尔哈撒韦]]一季度经营利润约80亿美元。他强调关注经营利润而非资本利得，因为股价短期波动难以预测，但长期投资一定会有回报。

经济方面，疫情期间大量货币投放导致供不应求，但过去六个月经济热度已消退。商家从"不愁卖"变成需要搞促销，很多经理人对此感到意外。[[巴菲特]]预期大多数公司今年利润不如去年。

### 投资收益与保险业务前景

[[巴菲特]]对两方面持乐观预期：

一是投资收益将大幅增加。[[伯克希尔哈撒韦]]持有约1250亿美元短期投资，一年多前短期国债收益率仅4个基点，现在可达5.92%，年收益从5000万美元跃升至约50亿美元。

二是[[保险业]]业务。[[保险业]]与经济周期无直接关系，主要受飓风、地震等灾难事件影响。从概率角度看，今年保险业务可能比去年好。

### 净资产与浮存金的价值

[[伯克希尔哈撒韦]]净资产5040亿美元，是美国最大。[[保险浮存金]]达1650亿美元，这是[[阿吉特·贾恩]]悉心经营的成果。[[巴菲特]]将[[保险浮存金]]比作"不用付利息、不担心挤兑的银行存款"，虽在账面显示为负债，实则是宝贵资产。

### 回购与留存收益策略

[[伯克希尔哈撒韦]]流通股数量逐年减少，股东不用额外出钱就能分得更多资产和利润。[[巴菲特]]坚持不分红，正是因为始终保留盈利再投资，才积聚起5000多亿美元净资产。1967年曾派发过一次每股0.10美元股息，[[巴菲特]]表示很后悔。

### 银行危机与政府应对

关于硅谷银行事件，[[巴菲特]]认为如果储户存款没得到全额赔付，后果将是灾难性的。联邦存款保险公司25万美元赔偿限额是死规定，但政府不可能死守条款让经济陷入混乱，全额赔付是必然的。

### 投资中的情绪管理

[[巴菲特]]表示，在[[伯克希尔哈撒韦]]历史上，从未因情绪化在投资中犯错。投资和商场上绝对不能受情绪影响，但生活中不能做没感情的人。唯一可能受情绪影响的是对待共事已久的老经理人，有时没能及早发现他们能力退化。

### 盖可保险的科技挑战

[[阿吉特·贾恩]]坦承盖可保险在远程信息处理技术方面落后于竞争对手。目前90%保单定价已纳入该技术，但不到一半投保人愿意接受。更大的问题是科技欠账太多——600多个系统无法互通，需压缩到15-16个。一季度综合成本率92.7%，但目标96%可能要到明年底才能实现，且会以投保人数减少为代价。

### BNSF铁路的效率提升

[[格雷格·阿贝尔]]表示为[[BNSF铁路]]团队感到骄傲。虽然竞争对手在使用"准确时刻铁路运输机制"，但BNSF团队每天都在追求更高效率，同时保证客户需求和安全。2019-2021年在提高效率方面成绩显著，2022年面临供应链中断等挑战后也在努力"重启铁路"。

## 原文金句

> "浮存金就像股东资金一样，也为资产提供资金来源。别的公司，没有像我们这样理解浮存金的。"

> "在投资中，在商场上，不能情绪化，不要受情绪影响。至于在生活中，那是另一回事。在生活中，不能做一个没感情的人。"

> "伯克希尔把所有的盈利留下来再投资，我们这才积聚起5000多亿美元的净资产，才能实现300多亿美元的经营利润。"

> "谁都不敢板上钉钉地说明天会怎样，明年会怎样。无论是市场涨跌，还是经济趋势，没一个预测是保准的。"
