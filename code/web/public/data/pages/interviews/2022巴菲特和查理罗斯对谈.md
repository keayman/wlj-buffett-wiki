# 2022巴菲特和查理罗斯对谈

## 核心要点

- 投资的本质是买企业而非买股票，关注公司10-20年后的价值而非短期股价波动
- 成功投资不需要天才智商，120点足够，关键是拥有正确的投资方向和心态
- 美国资本主义体系在过去300年创造了巨大财富，但需要政府作为规则制定者
- 核战争是最大的担忧，当前局势接近古巴导弹危机的紧张程度
- 91岁仍热爱工作，每天跳着踢踏舞去上班，被信任管理他人资产是最大的快乐

## 详细摘要

### 工作状态与团队

91岁的巴菲特身体状况极佳，搭档[[芒格]]已98岁仍老当益壮。继任者[[格雷格·阿贝尔]]已确定但尚未完全接手。巴菲特每天7点前起床关注新闻和市场动态，8点给团队下达当日交易指令，内容包括买卖比例、限价等，执行后无需复核。团队成员共事数十年，彼此了解信任。

### 投资哲学的形成

11岁时用114.75美元买了人生第一只股票——城市服务公司。前8年沉迷于技术分析和预测股价走势，直到19-20岁读了[[格雷厄姆]]的《聪明的投资者》才醒悟：买的不是股票，而是碰巧在公开市场交易的生意。从此不再关心股票短期涨跌，而是专注于理解企业本身。

### 投资心态与方法

90%的人买股票时想法不正确——希望买入后下周就涨。正确的思维是考虑公司10-20年后值多少钱，股价下跌时反而高兴，因为可以用同样的钱买更多。就像农民希望农田价格降低一样。投资不需要极高智商，120点足够，170点并不比120点做得更好，关键是方向正确。

### 关于动视暴雪投资

澄清动视暴雪的投资并非他本人决策，而是办公室里管理300亿美元的[[托德·库姆斯]]和[[泰德·韦施勒]]独立做出的。他们有权在不过问巴菲特的情况下进行投资。媒体报道的价格和时间都有误，实际77%在10月购买，成交价约77美元。

### 对美国经济与资本主义的看法

美国用300年时间，以不到1%的世界人口创造了超过20%的全球财富。当前美国人比以往任何时期都更富裕，银行存款增加，抵押贷款利率处于历史低位。资本主义需要政府作为规则制定者和裁判，但政府不应搞砸资本主义。改革从1789年就开始了，从未止步。

### 对未来的展望

美国经历过内战、大萧条、疫情，但一直在进步。核战争是最大的担忧，引用爱因斯坦的话：不知道第三次世界大战用什么武器，但第四次肯定是棍棒和石头。当前局势接近古巴导弹危机。尽管如此，仍相信未来会更好。

## 原文金句

> "我不是在买股票，而是在买非常优秀的美国企业。当股票下跌的时候我会很高兴，因为我可以用同样多的钱买更多的股票，就好比农民希望农田价格降低，这样就可以买到更多农田。"

> "做投资并不需要极度聪明的大脑。在这个游戏中，你可能需要120点的智商，但170的智商并不一定就比120的做得好。"

> "我买的不再是股票，而是只是碰巧在公开市场交易的生意。我成为这家公司的老板，我不在乎这只股票在第二天、下一周、下一个月或下一年是涨还是跌。"

> "联邦政府才是最后的老大，他们不应该搞砸资本主义，资本主义搞砸联邦政府却非常简单。"

> "我越来越笨，也越来越聪明了。"
