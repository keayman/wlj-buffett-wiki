# 2011 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]完成重要接班人布局，[[托德·库姆斯]]和[[泰德·韦施勒]]加入投资团队，CEO继任者已确定
- 五大非保险业务创纪录，税前收益合计超90亿美元，资本支出82亿美元创新高
- [[保险浮存金]]达706亿美元历史新高，连续9年实现承保盈利
- 重大投资：收购路博润、买入[[IBM]] 109亿美元、投资美国银行50亿美元
- 阐述[[回购]]的正确原则：必须以低于[[内在价值]]的价格进行

## 详细摘要

### 接班人规划与管理层

董事会在接班人规划上投入了大量时间和精力。2011年初[[托德·库姆斯]]加入，年底后[[泰德·韦施勒]]也加入团队。两人都具备杰出的投资才能，完全能够在[[芒格]]和巴菲特不再管理时接管整个投资组合。CEO继任者也已确定，还有两位出色的后备人选。巴菲特超过98%的净资产都投在[[伯克希尔哈撒韦]]股票上，对公司前景充满信心。

### 主要业务表现

五大非保险企业——[[BNSF铁路]]、伊斯卡、路博润、马蒙集团和[[中美能源]]——每一家都创下经营收益新纪录，税前收益合计超90亿美元。七年前只拥有[[中美能源]]一家，当时税前收益仅3.93亿美元。2011年资本支出达82亿美元，其中约95%投在美国国内。

### 保险业务与浮存金

[[保险业]]继续提供零成本资金。[[保险浮存金]]从410亿美元增长到706亿美元的历史新高。连续9年实现承保盈利，累计盈利约170亿美元。这意味着浮存金成本低于零。

### 重大投资决策

收购路博润，这是一家全球性特种化学品生产商。投资50亿美元购入美国银行6%优先股，附带认股权证。投资109亿美元购入6,390万股[[IBM]]股票。加上[[可口可乐]]8.8%、[[富国银行]]7.6%、美国运通13.0%，形成"四巨头"持仓。这四家公司归属于伯克希尔的收益份额达33亿美元，远超报表上8.62亿美元的股息收入。[[留存收益]]将在未来推动收益增长。

### 投资失误

购买未来能源控股公司[[债券]]约20亿美元是一个大错误。天然气价格暴跌导致累计减记近14亿美元。这是一个"重大非受迫性失误"——完全错误地估算了盈亏概率。

### 住房市场判断

去年预测"房地产复苏可能会在一年左右开始"是大错特错。[[克莱顿房屋]]等五家与住房相关的企业税前利润从2006年的18亿美元降至5.13亿美元。但住房市场一定会回来——每天新组建的家庭数量都超过新增住房数量。"虽然经济衰退期间，一些人最初的反应是和亲戚'合住'，但跟公婆挤在一起的吸引力很快就消失殆尽了。"

### 股票回购原则

只有同时满足两个条件才赞成[[回购]]：公司有充裕资金满足运营需求；股价相对于[[内在价值]]有很大折扣。仅仅说回购是为了抵消稀释或因为有多余现金，都站不住脚。"资本配置的第一定律——不管资金是用来[[收购]]企业还是回购股票——就是在一个价格上聪明的做法，在另一个价格上就是愚蠢的。"

以[[IBM]]为例说明长期股东应该期望什么：如果IBM用500亿美元回购股票，股价越低，能回购的股数越多，伯克希尔的持股比例就越高。因此长期股东应该期望股价在回购期间保持低迷。
