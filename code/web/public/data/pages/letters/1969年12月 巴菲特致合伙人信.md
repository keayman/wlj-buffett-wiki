# 1969年12月 巴菲特致合伙人信

## 核心要点

- 详细介绍多元零售公司和[[伯克希尔哈撒韦]]两家控股公司的业务构成和财务状况
- 坦率评价旗下业务：三项一流业务（保险、银行、联合零售）和一项平庸业务（纺织）
- 强调这些股票应被视为"公司"而非"股票"，适合长期持有
- 说明合伙基金清算后的股票分配安排及交易限制
- 预计1969年整体收益率略高于6%

## 详细摘要

### 多元零售公司的业务重组

多元零售公司流通股100万股，巴菲特合伙基金持有80万股。公司已将霍赫希尔德·科恩公司出售，获得约1100万美元现金（含票据现值）。目前保留的联合零售商店净资产约750万美元，财务稳健、利润率高，去年销售额3750万美元，净利润约100万美元，今年预计净利润可达110万美元。

公司有形资产净值每股11.50-12.00美元，拥有大量闲置资金可用于投资。发行的660万美元公司债包含特别条款：若巴菲特不再是最大股东，债券持有人可要求按面值偿付。

纺织业务每股占16美元资产，但资本回报率低，"算不上好生意"。巴菲特坦言四年半前买入时纺织是最好的业务，现在只要能维持就继续经营。

[[保险业]]务（[[国民保险公司]]100%股权）和银行业务（97.7%股权）则是"一流的好生意"，资本回报率高。银行每股约17美元，保险每股约15美元，合计32美元，估计正常盈利能力每股4美元，成长前景非常好。

公司账面净资产每股45美元，但需注意保险和银行持有的7500万美元[[债券]]按摊销成本计价，市场价值低于账面约每股10美元。

### 管理层与[[企业文化]]

四大主要业务中三项一流，经理人都超过60岁，基本白手起家。他们"工作努力、生活富足、人品很好"。巴菲特愿意投入大部分资金的重要原因就是有这些优秀的人经营公司。

### 股息政策与[[内在价值]]展望

两家公司目前不发放股息，未来几年可能仍然如此。原因包括：偿还负债、保护储户和投保人、追加投资获取回报、寻找新业务扩展盈利。

巴菲特认为多年后两家公司[[内在价值]]会实现巨大增长，若增长率达不到每年10%会很失望。股价围绕内在价值波动，但长期总会反映价值。

### 清算安排与持股限制

1月5日现金分配预计占1969年1月1日资本的64%以上，若蓝筹印花如期上市可达70%以上。股票分配后存在交易限制，需遵循律师指导意见出售。

巴菲特明确：分配后与合伙人不存在管理或合伙关系，不做无限期提供建议的承诺，可能只参与重大决策，甚至成为被动股东。

## 原文金句

> "在我眼里，它们是公司，不是'股票'。只要公司长期经营得好，股价也没问题。"

> "虽说与整个纺织行业相比，伯克希尔的纺织业务稍微领先，但这项业务算不上好生意。"

> "股票的市场价格围绕内在价值剧烈波动，但是，长期来看，市场价格总有反映内在价值的时候。"

> "我之所以愿意把我的大部分资金投入到伯克希尔哈撒韦和多元零售公司，其中很重要的一个原因就是，有这么优秀的人经营这两家公司。"
