# 2008巴菲特答Emory与Austin大学商学院学生问巴菲特答

## 核心要点

- 职业投资者应高度集中投资，普通人应高度分散并定投指数基金
- 好生意的标准是拥有宽广的[[护城河]]，能经受住竞争考验
- 选择经理人的核心是看他们是否热爱自己的生意，而非金钱
- 投资成功的关键在于性格和耐心，而非智商或学历
- 避免使用[[杠杆]]，不把命运交到别人手里

## 详细摘要

### 关于[[分散投资]]与集中投资

巴菲特对[[分散投资]]持两种观点：职业投资者如果对自己有信心，应该高度集中；普通人不懂投资，应该高度分散，购买低手续费的指数基金并长期定投。他和[[芒格]]主要投资5只股票，愿意把25%仓位分配给第一重仓股。历史上他曾用40%仓位投资美国运通，用75%仓位投资长期资本管理公司危机中的国债套利机会。

关键原则是：不把资金投到第20位的机会而放弃第1位的机会。50-60年来，他们没有一笔投资让他们亏损超过个人总资产的2%。

### 关于好生意与[[护城河]]

巴菲特不预测哪个行业最具成长性。他回顾了汽车和航空业的历史——给社会带来巨大贡献，却给投资者带来巨大痛苦。好生意的标准是：产品始终不变、拥有[[护城河]]、经得起竞争考验。

[[可口可乐]]122年来每天卖出15亿瓶，[[吉列]]占剃须刀70%市场份额且毛利率80%。这些公司是"打不倒的"，即使营销天才也无法撼动。

### 为什么[[格雷厄姆]]的方法少有人模仿成功

巴菲特19岁读了《聪明的投资者》，这本书改变了他的一生。但[[格雷厄姆]]的理念能听进去的人一听就懂，听不进去的人怎么讲都没用。归根结底在于性格差异——大多数人想要赚钱快，想要刺激，而价值投资要求坚守能力圈、等待明显的好机会。

### 关于选择经理人

[[伯克希尔哈撒韦]]有45位经理人，选择标准是看他们是否热爱自己的生意。关键问题是：有了5亿美元之后，这位经理人是否还能早上6点起来工作。巴菲特要盯着他们的眼睛，看他们爱的是生意还是金钱。

他提到了[[内布拉斯加家具店]]的[[B夫人]]作为典范。从1965年至今，没有一个经理人转投竞争对手。

### 关于人生榜样与成功

巴菲特认为自己的成功与个人努力关系不大——他生在1930年的美国，天生有资产配置的能力，生在了天赋大有用武之地的时代。他的父亲和[[格雷厄姆]]是最重要的榜样。

他强调：找到好伴侣就成功了一大半，身体健康、家庭幸福，一辈子一定能过得很好。

### 关于寻找投资机会

巴菲特喜欢"开枪打桶里的鱼，而且最好是桶里没水了再开枪"。他曾翻阅《穆迪手册》一页一页寻找机会，在第1443页找到了西部保险证券——每股盈利21-29美元，股价却在3-13美元之间。

机会不需要太多，一生找到10个足够发达。关键是不能犯大错——再大的数乘以零，结果都是零。
