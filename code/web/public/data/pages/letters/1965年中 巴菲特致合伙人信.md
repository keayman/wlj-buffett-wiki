# 1965年中 巴菲特致合伙人信

## 核心要点

- 上半年合伙基金收益率10.4%，领先道指9.6个百分点，但业绩表现与预期模式相反——市场上涨时领先，下跌时跟随
- 通过一系列买入活动取得了一只"低估类"股票的控股权，从此评判价值的首要决定因素变成企业的资产和盈利能力
- 基金公司如同"漂在池塘上的鸭子"，水涨则涨、水落则落，无法证明其选股能力优于随机选股
- 设立预先存取资金机制，6%利率双向适用，为合伙人提供流动性便利

## 详细摘要

### 上半年业绩与预期偏离

1965年上半年道指整体收益率仅0.8%，合伙基金未扣除分成的收益率达10.4%。但实际表现与预期完全相反：按照既定策略，合伙基金应该在市场下跌时领先、在泡沫中落后。然而这半年，市场大涨时合伙基金遥遥领先，后来市场下跌时却跟随道指下跌。

巴菲特坦言对此并不引以为傲，如实报告是因为合伙人有权知道实情，也说明短期情况可能远远偏离预期。他提醒合伙人：与平均水平的偏离不总是偏好，偏向差的一面时肯定会有。

### 控股投资的转变

通过一系列买入活动，合伙基金取得了一只[[低估]]类股票的控股权。取得控股权后，评判价值的首要决定因素从供求关系变成了企业的资产和盈利能力。巴菲特强调：持有少数权益时，盈利能力和资产只能间接影响价值；但取得控股权后，拥有的不再是股票而是公司，要按公司本身估算[[内在价值]]。

他对[[收购]]成本和企业运营都非常满意，也非常看好公司的管理层。

### 基金公司的"鸭子理论"

巴菲特将基金公司比作漂在池塘上的鸭子——水涨则涨、水落则落。1965年上半年池塘水位几乎没变，四大基金公司业绩也原地踏步。他引用芝加哥大学证券价格研究中心主任詹姆斯·洛瑞的话："我们找不到证据，无法证明基金公司的选股能力优于随机选股。"

巴菲特表示，无论潮起潮落都将矢志不渝地搏浪击水。要是业绩下降还不如让合伙人自己在水面上漂浮，只能引咎辞职。

### 预先存取资金机制

合伙基金接受预先存入资金，支付6%利息；也允许预先提取最高20%的资金，收取6%利息。存贷利率相同看似不符合商业逻辑，但巴菲特认为这符合所有合伙人利益：合伙人间接持有大量流动资产，应该提供一定流动性；同时预先存入的资金能成为可预见的权益资本，投资心态与短期银行贷款不同。
