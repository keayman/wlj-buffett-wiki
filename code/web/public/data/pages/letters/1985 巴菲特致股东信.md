# 1985 巴菲特致股东信

## 核心要点

- 伯克希尔净资产增长48.2%，但这种增长率在未来无法持续，原因是市场缺乏低估机会以及公司规模过大
- 完成多项重大交易：买入大都会/ABC股份、收购斯科特费泽、出售通用食品股票
- 关闭经营21年的纺织业务，深刻反思了在衰退行业持续投资的错误
- 机构投资者的羊群行为反而使市场更不理性，而伯克希尔的个人股东群体创造了稳定合理的股价

## 详细摘要

### 规模与回报率的困境

1985年伯克希尔净资产增加6.136亿美元，增幅48.2%。自现任管理层接手21年以来，每股账面价值年复合增长率约23.2%。但[[芒格]]和我认为这种增长率无法持续，原因有二：

第一，当前股票市场提供的投资机会远不如十年前。那时唯一的问题是挑哪个便宜货，现在是根本没有便宜货。我们保险公司持有的都是拥有卓越商业特质和杰出管理层的公司，但市场价格已充分反映这些优点，不可能再取得过去那样规模的收益。

第二，规模的增长终将拖累卓越的经济回报。看看那些高回报率的公司，一旦股本规模超过10亿美元，没有一家能在随后十年里把全部收益进行再投资的同时维持20%以上的股本回报率。去年我们需要赚39亿美元才能达到15%年回报率，如今门槛已提高到57亿美元。

### 股价与[[内在价值]]的关系

从历史上看，[[伯克希尔哈撒韦]]的股价一直略低于其[[内在价值]]。但最近这种折价已消失，偶尔甚至出现小幅溢价。我个人更倾向于市场价格始终贴近企业[[内在价值]]——在这种关系下，所有股东在持有期间的回报都能与公司业绩精准对应。

伯克希尔的市值与[[内在价值]]之间的关系，比我所知道的任何其他上市公司都要稳定。这要归功于各位股东——我们几乎所有股东都是个人投资者，而非机构。你也许会认为机构应该成为金融市场稳定与理性的力量，事实恰恰相反：那些被机构大量持有的股票，其价格往往是最不合理的。

[[格雷厄姆]]40年前讲过一个故事：一位石油勘探商去世后来到天堂，发现石油商人专区已满。他大喊"地狱里发现石油了！"所有石油商人争先恐后朝地狱奔去。圣彼得邀请他搬进去，他却说："不了，我还是跟他们一起去吧。万一那传言是真的呢。"

### 纺织业务的关闭与教训

7月份我们决定关闭纺织业务。[[芒格]]一直强调研究失败比研究成功更重要。

21年前收购[[伯克希尔哈撒韦]]控制权时，公司账面净资产2,200万美元全部投在纺织业务上，但[[内在价值]]远低于此。我们选中肯·蔡斯担任总裁，这个决定百分之百正确。1967年纺织业务产生的现金被用来收购[[国民保险公司]]，进入[[保险业]]。

我们之所以继续保留纺织业务，是因为：它是当地重要雇主、管理层坦诚积极、员工理解配合、相对于投入资本应能产生适度现金回报。但最后一点我判断大错特错——1979年后纺织业务便开始大量消耗现金。

多年来我们面临追加大量资本支出以降低可变成本的抉择。每个投资方案看起来都稳赚，但纺织投资所许诺的收益只是幻象。我们的竞争对手都在进行同样投资，一旦足够多的公司这样做，降低的成本就变成全行业降价的基准——就好比每个看游行的人都觉得踮起脚尖就能看得更清楚一样。

### 主要投资交易

1985年已实现资本利得中有很大一部分来自出售通用食品股票。我们从1980年起便持有，[[买入价格]]远低于每股商业价值。我们受益于四个要素：便宜的[[买入价格]]、优秀的[[商业模式]]、专注于股东利益的能干管理层，以及愿意支付完全商业价值的买家。在选择股票时，我们把全部精力放在漂亮地买进上，而不是漂亮地卖出。

我们还收到了来自[[华盛顿邮报]]和通用食品的大额特别分红，这是在公司[[回购]]其他股东股份的同时，我们按比例出售部分持股时产生的。我们认为不卖出的股东因公司以低于[[内在价值]]的价格[[回购]]股份而受益。
