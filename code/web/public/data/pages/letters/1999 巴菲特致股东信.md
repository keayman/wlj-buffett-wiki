# 1999 巴菲特致股东信

## 核心要点

- 1999年是巴菲特任期内表现最差的一年，每股账面价值仅增长0.5%，主要归咎于资本配置失误和股票投资组合表现不佳
- [[浮存金]]是评估保险公司的核心指标，伯克希尔浮存金达253亿美元，但成本从接近零飙升至5.8%
- 盖可保险在托尼·奈斯利领导下取得非凡进步，有效保单从231万增至433万，计划大幅加码营销投入
- [[阿吉特·贾恩]]白手起家建立了卓越的再保险业务，持有63亿美元浮存金且实现承保盈利
- 伯克希尔的经理人文化独特：不签合同、不做预算汇报、把公司当成要传承百年的家族资产来经营

## 详细摘要

### 业绩反思与前景展望

1999年[[伯克希尔哈撒韦]]净资产仅增加3.58亿美元，无论绝对数字还是相对标普500指数，都是巴菲特35年任期内最差的一年。罪魁祸首是[[资本配置]]失误——股票投资组合表现糟糕，几笔大型投资标的因自身经营不佳而远远落后大盘。

尽管如此，巴菲特和[[芒格]]仍预期伯克希尔未来十年的[[内在价值]]增长将略微超越标普500。但他坦言，大幅超越指数的辉煌已成过去——当年资本规模小、价格便宜，投资机会比现在宽广得多。他还预判标普500未来一二十年的表现将远不如1982年以来那段岁月。

### 内在价值的衡量

巴菲特提供了估算伯克希尔内在价值的两个关键指标：每股投资规模和税前每股经营收益。1999年每股投资达47,339美元，但经营收益因[[通用再保险]]巨额承保损失而转负。他强调，[[商誉]]摊销是纯会计事项，与真正的经济商誉无关——像[[喜诗糖果]]和[[布法罗新闻报]]这样的业务，实际价值约为账面价值的15到20倍。

### 独特的经理人文化

伯克希尔的经理人队伍与众不同：很多人已经非常富有，继续工作纯粹因为热爱。公司不签合同、不做预算汇报、没有资本支出限制，只要求经理人把公司当成要传承百年的家族资产来经营。

[[威利家居]]的比尔·查尔德是典型代表。这位摩门教徒坚持周日不营业，却把年营业额从25万美元做到3.42亿美元。当他提议在博伊西开分店时，主动提出自掏腰包900万美元买地建店——如果失败，他一人承担损失。分店大获成功后，他按成本价移交物业，甚至拒绝收取两年的资金利息。

### 保险业务的核心逻辑

[[保险业]]是伯克希尔最主要的业务。评估保险公司的三个关键因素：[[保险浮存金]]规模、取得成本、以及长期前景。浮存金是保费收取与理赔支付之间的时间差产生的资金，保险公司可用于投资。

1999年伯克希尔浮存金达253亿美元，但承保损失14亿美元，成本飙升至5.8%。[[阿吉特·贾恩]]的贡献最大——他白手起家建立卓越的再保险业务，持有63亿美元浮存金且实现承保盈利。阿吉特身上罕见地兼具四项特质：用智慧合理定价、用理性放弃无法评估的风险、在保费合适时勇于承接巨额保单、在保费不足时严守[[承保纪律]]。

盖可保险在托尼·奈斯利领导下取得非凡进步，有效保单从1995年的231万增至1999年的433万。营销支出从3,300万美元增至2.42亿美元，电话顾问从652人增至2,631人。巴菲特表示，只要最后一块钱投入还能以有吸引力的成本带来新客户，他们很乐意每年拿出10亿美元投入营销。

### 收购动态

1999年伯克希尔买下乔丹家具，并签约收购[[中美能源]]大部分股权。两笔交易全部以现金完成，没有发行任何新股——这是巴菲特和芒格最偏爱的方式。
