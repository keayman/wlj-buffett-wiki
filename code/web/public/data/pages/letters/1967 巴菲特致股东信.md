# 1967 巴菲特致股东信

## 核心要点

- 纺织业务面临市场萎缩和价格下跌双重压力，但第四季度出现回暖迹象
- 收购保险子公司是实现盈利来源多元化的重要一步，其收益已超过资本投入更大的纺织业务
- 关闭精梳棉面料工厂的决定体现了对市场趋势的务实判断——当产品需求消失时，效率再高也无济于事
- 管理层明确表态将积极寻求扩张机会，目标是实现与动用资本相称的稳定盈利

## 详细摘要

### 纺织业务的困境与调整

1967财年销售额从4,940万美元降至3,900万美元，原因是需求萎缩和价格下跌。家用纺织品部门表现出最强抗压能力。为避免库存积压，公司削减了约15%的产能，但这导致熟练技工流失，未来恢复满负荷运转时将面临较高培训成本。

公司采取了产品多元化策略：拓展家用纺织品产品线，新设服装面料部门进军女装市场。这一策略的核心是在与主流生产商的正面竞争之外，开辟属于自己的产品线。

### 精梳棉面料业务的终结

市场调查显示，消费者对纯棉精梳面料的需求正被涤棉混纺面料取代。公司决定关闭位于罗得岛州沃伦市的菲利普国王工厂D分厂。这座工厂生产效率不低，但市场需求已近乎消失，设备改造在经济上不合算。预计税前损失不超过100万美元。

### [[保险业]]的成功布局

1967年3月完成收购的[[国民保险公司]]和国民火灾与海事保险公司表现令人满意。净保费收入可观提升，承保利润率良好，投资收益大幅增加。保险子公司的全部收益均予以留存，以增强资本实力。

这次投资是公司实现盈利来源多元化的第一个重要步骤。1967年的数据证明了其价值：保险子公司创造的收益大幅超过资本投入规模更大的纺织业务。拥有两个互不相关的收益来源，比仅依赖单一行业更具优势。

### 未来战略方向

管理层将持续关注扩张机会，无论是纺织业、[[保险业]]还是其他领域。公司持有的流动资产以可随时变现的普通股为主，可用于[[收购]]或把握现有业务的盈利机会。若有吸引力的机会，公司不会对借款犹豫。目标明确：实现与动用资本相称的稳定而可观的盈利能力。
