# 1995 巴菲特致股东信

## 核心要点

- 完成三笔梦寐以求的并购：赫兹伯格钻石店、[[威利家居]]和盖可保险，营收翻倍但股份和负债几乎未增
- 双管齐下的投资策略：协商买下整家公司，或通过股市买入部分股权，比单一路径的资金配置者更具优势
- 零售业是"每天都得聪明"的生意，与"只需聪明一次"的生意形成鲜明对比
- [[伯克希尔哈撒韦]]的并购优势：没有战略规划束缚、可提供稳定持久的家、经理人享有非凡自主权

## 详细摘要

### 年度业绩与投资哲学

1995年净资产增加53亿美元，增幅45%，每股账面价值增长43.1%。31年来每股账面价值从19美元成长到14,426美元，年[[复利]]增长率约23.6%。但巴菲特坦言在去年的股票市场里"随便一个笨蛋都能大赚一笔"，水涨船高——只不过涨的全是游艇。

巴菲特与[[芒格]]致力于打造由经济特性优异、杰出经理人领导的企业集团。他们采用双管齐下的做法：通过协商以合理[[买入价格]]取得整家公司，或通过股市以低于等比例价格买入优秀企业的部分股权。这种折衷主义使他们比坚持单一路径的资金配置者拥有重要优势。

### 对并购的深刻反思

大部分并购案会损害收购方股东利益。卖方及其代表总是端出娱乐价值远高于教育价值的财务预测，华尔街画大饼的本事一点不输华盛顿。巴菲特从不看卖方准备的预测数据，因为卖方对自家业务的了解几乎总是远超买方，而且会挑选"马走路正常"的时候卖出。

[[伯克希尔哈撒韦]]的并购优势在于：没有战略规划，不必沿既定方向以离谱价格[[收购]]；做决策时总是将交易与其他几十种机会比较，包括通过股市买入世界上最好企业的部分股权。此外，伯克希尔股票可作为支付对价，让卖方无限期递延个人税负，同时为公司找到稳定持久的家。

### 三笔重要并购

赫兹伯格钻石店源于一次"走动式并购"——巴菲特在纽约街头被股东小巴内特·赫兹伯格拦下。这家1915年创立的公司已发展成134家门店的连锁集团，营业额从1974年的1,000万美元增长到1994年的2.82亿美元。单店年均营业额约200万美元，远超竞争对手，这是其卓越盈利能力的关键。

[[威利家居]]的并购由[[内布拉斯加家具店]]的艾文·布鲁姆金牵线。这家犹他州最大的家居用品商，在CEO比尔·蔡德的带领下，从1954年的25万美元营业额增长到1995年的2.57亿美元，占据犹他州超过50%的家具市场份额。

### 零售业的残酷本质

零售业是一门艰苦的生意。许多零售商在一段时期内享受惊人成长和超高股本回报率，然后突然急转直下，甚至滑进破产深渊。零售商必须每天都保持聪明——竞争对手随时在模仿并超越你，消费者被各种花样吸引去尝试新商家。一旦松懈就注定失败。

与此形成对比的是"只需聪明一次"的生意，比如早期买下地方电视台，即使放个懒惰无能的侄子去管，照样能红火几十年。但零售业用那个侄子就等于买了张直通破产的快车票。

### 盖可保险的45年情缘

巴菲特与盖可保险的渊源始于1950年在哥伦比亚大学追随[[格雷厄姆]]学习时。他发现格雷厄姆是盖可保险董事长，便在1951年1月的一个周六前往华盛顿总部，遇到了总裁助理洛里莫·戴维森，接受了一场四小时的[[保险业]]速成课程。

戴维森说明了盖可保险直销模式带来的巨大成本优势——竞争对手通过代理人销售的分销模式已根深蒂固，根本不可能放弃。1951年巴菲特以10,282美元买入350股盖可保险，年底市值13,125美元，超过个人净资产的65%。可惜1952年他以15,259美元清仓换股，而这些股份在接下来20年增值到约130万美元——这给他上了一课：卖掉已确认是卓越企业的股份是多么不明智。
