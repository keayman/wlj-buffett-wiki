# 2025年伯克希尔股东大会

## 核心要点

- [[伯克希尔哈撒韦]]持有3350亿美元短期国债，但巴菲特强调这不是刻意为之，而是因为极度低估的机会可遇不可求
- 日本五大商社投资是长期承诺，未来五十年甚至更长时间都不会卖出，并期待更深层次的合作
- 贸易应促进全球共同繁荣，不应被当作武器使用，美国的成功不应建立在其他国家的损失之上
- 好的投资机会没有规律可循，关键是耐心等待并在机会出现时重拳出击

## 详细摘要

### 股东大会盛况

本届股东大会创下多项纪录：19,700名股东参与购物活动，[[喜诗糖果]]创造317,000美元销售额，布鲁克斯跑鞋销售310,000美元。巴菲特特别感谢了为大会付出努力的[[伯克希尔哈撒韦]]总部员工，以及编写六十周年纪念书籍的凯丽·索娃。

巴菲特还特别感谢了苹果公司CEO蒂姆·库克的到场，称赞乔布斯创立苹果的壮举无人能及，而库克发展壮大苹果同样了不起。

### 关于贸易与关税

巴菲特区分了他2003年提出的进口证制度与当前的关税政策。进口证旨在平衡进出口、遏制贸易赤字，同时促进发展中国家经济进步。他强调贸易平衡有利于全世界，各国应发挥各自优势进行分工协作。

巴菲特警告不要将贸易当作武器："3亿人自鸣得意，全世界的75亿人侧目而视，这绝对不是我们想要的。既有失道义，也缺乏智慧。"他认为全世界过得越好，美国也会过得越好、更有安全感。

### 日本五大商社投资

约六年前，巴菲特翻阅股票手册时发现了五家日本综合商社，股价极为低廉。[[伯克希尔哈撒韦]]花了一年时间买入，目前持股接近10%的约定上限，正在商谈提高持股比例。

[[格雷格·阿贝尔]]表示每年与五家商社交流两三次，希望培育长期合作关系，期待未来能携手在全球范围内做出大手笔投资。巴菲特承诺未来五十年不会卖出这五家公司的股票。

[[可口可乐]]、苹果、[[美国运通]]等[[伯克希尔哈撒韦]]投资的公司在日本都有大量业务。巴菲特提到在日本，[[可口可乐]]最畅销的产品是乔雅咖啡，体现了尊重当地市场特性的态度。

### 现金配置策略

[[伯克希尔哈撒韦]]目前持有3350亿美元短期国债，占总资产27%。巴菲特解释这并非刻意为接班做准备，而是因为好的[[买入价格]]机会不是想有就有的。他提到差点做成一笔100亿美元的投资，如果机会合适，可以投出1000亿美元。

[[芒格]]曾批评巴菲特"太多动了"，认为一辈子做50笔投资不如做5笔。巴菲特强调投资的关键是等待机会，而非强行每年投出固定金额。[[伯克希尔哈撒韦]]每年产生约400亿美元自由现金，但极度[[低估]]的机会可遇不可求。

## 原文金句

> "3亿人自鸣得意，全世界的75亿人侧目而视，这绝对不是我们想要的。既有失道义，也缺乏智慧。"

> "全世界过得越好，我们过得越好，我们更有安全感，我们的子女也更有安全感。不是说，别的国家过得好了，我们就吃亏了。"

> "一页一页地翻，总会有意外收获。要做好投资，少不了翻过每一页的功夫。"

> "极度低估的机会可遇而不可求。虽然长期趋势向上，但没人知道明天、后天、下个星期、下个月，市场走势如何。"
