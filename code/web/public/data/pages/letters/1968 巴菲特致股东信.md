# 1968 巴菲特致股东信

## 核心要点

- 纺织业务和保险业务盈利均有改善，但整体股东权益回报率仍不理想
- 决定退出亏损的梭箱织机坯布业务，体现理性资本配置原则
- 保险业务在行业普遍亏损时仍实现承保利润，坚持"绝不为规模牺牲利润率"
- 通过收购太阳报业正式进入出版业，展现多元化扩张战略

## 详细摘要

### 纺织业务：有进有退的务实决策

家居纺织品和男装衬里业务表现良好，营业额增长约14%。1969年将引进新型织机以丰富产品线，并保留扩产期权。

然而，梭箱织机部门多年经营困难重重——偶尔的好时光总被进口冲击、价格战和产能压缩所打断。经过审慎评估，决定在1969年逐步退出该业务。虽然固定资产出售会带来一定亏损，但预计纺织业务整体状况将因此改善。

### 保险业务：逆势盈利的承保纪律

[[国民保险公司]]在行业普遍承保亏损的背景下，依然实现了良好的承保利润。投资收益大幅增加，既得益于资产规模扩大，也受益于[[债券]]收益率提升，普通股投资也兑现了部分资本利得。

核心原则是：[[承保纪律]]必须坚守，绝不为追求规模而牺牲利润率。由于坚持将保险子公司的[[留存收益]]全部保留，资本金大幅充实，承保能力随之增强。

### 投资与收购：为更强盈利能力布局

1968年出售部分有价证券，税后获利约150万美元，年底剩余持仓未实现利得达640万美元。这些证券作为临时性投资持有，随时可为[[收购]]或扩大经营调配资金。

年底后收购了太阳报业公司和布莱克印务公司，正式进入[[媒体与出版]]业。此次收购仅动用了可用资金的一小部分，规模虽小但增长潜力可观。
