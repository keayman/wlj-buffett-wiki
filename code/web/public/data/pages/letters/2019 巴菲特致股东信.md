# 2019 巴菲特致股东信

## 核心要点

- GAAP新规要求将股票未实现损益计入当期收益，导致报告利润剧烈波动，投资者应关注经营收益而非账面损益
- [[留存收益]]的复利效应是长期投资回报的核心驱动力，被投资公司保留的收益虽不计入报表，却在持续创造价值
- [[保险浮存金]]达到1,294亿美元，这些"先收后付"的资金为[[伯克希尔哈撒韦]]提供了独特的投资优势
- [[收购]]如同婚姻，追求阶段容易眼冒星星，现实往往与期望不符——好在"差"生意会萎缩，"好"生意会持续增长

## 详细摘要

### 会计规则与真实收益的背离

2019年[[伯克希尔哈撒韦]]GAAP收益814亿美元，其中537亿来自股票未实现收益增长。[[芒格]]和巴菲特都不认同2018年实施的新规则——它要求将股票组合的每一次波动都计入损益。2018年股市下跌导致报告收益仅40亿，2019年股市上涨则暴增至814亿，两年间GAAP收益疯狂波动1,900%。但在现实世界中，[[伯克希尔哈撒韦]]的股票持仓[[内在价值]]一直在稳步增长。

### [[留存收益]]的力量

1924年埃德加·劳伦斯·史密斯的著作揭示了一个"新奇"道理：管理良好的公司会保留部分利润再投资，产生[[复利]]效应。凯恩斯称赞这一洞见，但其实卡耐基、洛克菲勒早就靠此致富。

[[伯克希尔哈撒韦]]十大持仓公司中，2019年收到股息38亿美元，但属于伯克希尔的[[留存收益]]份额高达83亿美元——这些钱不会出现在报表上，却在勤奋地创造增值。持仓包括[[可口可乐]]、[[富国银行]]、苹果、美国运通等。

### 非保险业务的"婚姻"哲学

汤姆·墨菲的忠告："要想赢得好经理人的名声，关键是要收购好企业。"[[收购]]记录好坏参半，有些是彻头彻尾的灾难，有些则超出期望。幸运的是，"差"生意趋于停滞、占用资本越来越少，"好"生意持续增长。

[[BNSF铁路]]和[[伯克希尔哈撒韦能源]]是两大领头羊，2019年合计收益83亿美元。[[克莱顿房屋]]、[[利捷航空]]等其他业务表现稳健。非保险业务总净收益177亿美元。

### [[保险业]]的独特优势

1967年以860万美元收购[[国民保险公司]]以来，[[保险业]]一直是增长引擎。"先收后付"模式产生的[[保险浮存金]]从1970年的3,900万增长到2019年的1,294亿美元。

低利率环境下，多数保险公司从浮存金只能赚2-3美分，而[[伯克希尔哈撒韦]]凭借雄厚资本和多元收益流，拥有远超同行的投资灵活性。过去17年有16年实现[[承保纪律]]盈利。

## 原文金句

> "查理和我建议大家关注经营收益，同时忽略季度和年度的投资损益，不论是已实现的还是未实现的。"

> "要想赢得好经理人的名声，关键是要收购好企业。"

> "收购很像一场婚姻。当然，开头都是一场欢天喜地的婚礼——然而此后的现实往往与婚前的期望大不相同。"

> "随着时间推移，'差'生意往往趋于停滞，而我们的'好'生意往往会持续增长，并找到以有吸引力的回报率投入更多资本的机会。"
