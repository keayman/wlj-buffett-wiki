# 1963 巴菲特致合伙人信

## 核心要点

- 衡量业绩的标准是相对道指的领先幅度，而非绝对收益——即使下跌20%但跑赢道指10个百分点，仍是丰收之年
- 三类投资方法各有特点：低估类靠价格便宜取胜，套利类靠确定性和时间表获利，控制类靠取得话语权实现价值
- 复利的威力惊人，收益率相差几个百分点，长期结果差距成倍放大
- 公募基金整体难以跑赢道指，因为机构框架内很难特立独行
- 对未来保持清醒：当前的高收益率和大幅领先优势难以长期持续

## 详细摘要

### 业绩回顾与衡量标准

1963年合伙基金取得38.7%的收益率，跑赢[[道琼斯指数]]17.7个百分点。但巴菲特强调，丰收的定义不在于绝对收益，而在于相对基准的领先。七年累计，有限合伙人收益率达311.2%，年化22.3%，而道指同期仅95.1%，年化10.0%。

巴菲特坦言，长期保持如此大的领先优势是不可能的。能持续跑赢道指10个百分点就已经很了不起，必须做好某些年份明显落后的思想准备。

### 公募基金的困境

四家最大的基金公司（合计管理约40亿美元）的业绩与道指基本持平甚至略逊。这说明道指并非容易战胜的基准。华尔街精英们收取高额费用，却连打平指数都做不到。

巴菲特认为这不是批评，而是现实：在机构框架内管理巨额资金，想特立独行极其困难。公募基金的价值在于操作简单、自动[[分散投资]]、帮助投资者远离错误方法，而非跑赢市场。

### [[复利]]的威力

以达芬奇《蒙娜丽莎》为例：1540年弗朗西斯一世花20,000美元购入，若以6%年化复利投资至今，将值1000万亿美元。这个故事说明两点：活得越长越好；收益率哪怕相差几个百分点，最终差距惊人。

10万美元在不同收益率下30年的结果：4%变22.4万，8%变90.6万，12%变289.6万，16%变848.5万。只要合伙基金能适度跑赢主流投资工具几个百分点，长期就能创造惊人回报。

### 三类投资方法

**[[低估]]类**：以定量分析为主判断股票被低估，公司市值远低于产业资本的合理估值。这类股票不光鲜、没追捧，唯一优势是价格低廉。与道指走势高度同步，牛市表现最佳。

**[[套利]]类**：有时间表可循，机会来自公司行动（出售、并购、重组等）。只看公开公告，不依赖传闻。主要风险是意外导致交易落空。确定性高、持有时间短，年化收益可观。熊市积累领先优势，牛市拖累整体业绩。

**控制类**：通过大宗交易或持续买入取得控股地位。一旦控股，投资价值取决于企业本身而非[[市场先生]]的报价。需要几年时间衡量回报，要求利润空间必须足够大。

这种灵活性让合伙基金进退自如：市场回心转意则股价上涨；市场视而不见则继续买入直至控股。

### 规模与其他事项

初始净资产达1745万美元。巴菲特认为规模增长目前未稀释业绩，投资机会总比账户里的钱多10%。他与苏茜在基金中投资239万美元，加上大家庭成员合计超过360万美元，与合伙人共进退。

## 原文金句

> "就算我们跌了20%、道指跌了30%，我这封信还是会以'1963年是丰收的一年'开头。"

> "我们有意愿、也有财力去取得控股权，这使我们在低估类的很多持仓上都具备了进退自如的灵活性：如果市场回心转意，股价自然会上涨；如果市场仍然视而不见，我们就继续买入，直到公司成为我们的，届时评判我们判断正确与否的，是公司本身，而不是市场。"

> "在机构的框架内管理数十亿美元的资金，能取得的也只能是这样的业绩。在这套体系里，想特立独行是极其困难的。"

> "收益率相差区区几个百分点，对长期复利投资的最终成果影响极为巨大，而且时间越长，这种差距越是成倍放大。"
