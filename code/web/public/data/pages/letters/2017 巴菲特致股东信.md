# 2017 巴菲特致股东信

## 核心要点

- 2017年[[伯克希尔哈撒韦]]净资产增加653亿美元，但其中290亿美元来自税改红利而非经营业绩
- 新会计准则将未实现投资损益纳入净收益，会严重扭曲报表数字，投资者需关注经常性盈利能力而非GAAP净利润
- 收购市场价格创历史新高，[[伯克希尔哈撒韦]]坚持纪律，别人越轻率我们越谨慎
- [[保险浮存金]]达到1,145亿美元的历史高位，这是驱动[[伯克希尔哈撒韦]]增长51年的核心引擎
- 对[[杠杆]]的厌恶是长期原则：为得到不需要的东西而拿已拥有的去冒险是疯狂的

## 详细摘要

### 税改与会计准则变更

2017年[[伯克希尔哈撒韦]]净资产增长653亿美元，每股账面价值增长23%。但这并非寻常年份——360亿美元来自经营，290亿美元是税改带来的一次性收益。

新GAAP准则要求将未实现投资损益纳入净收益。[[伯克希尔哈撒韦]]持有1,700亿美元有价证券，一个季度波动100亿美元很正常，这会完全淹没真正重要的经营数字。巴菲特强调，投资者应关注经常性每股盈利能力的提升，而非被会计数字误导。

### 收购活动与市场环境

[[收购]]市场出现狂热。品质不错但远谈不上惊艳的企业，[[买入价格]]已创历史新高。原因包括：CEO职位自带"敢闯敢干"筛选机制、下属憧憬更大管辖范围、投资银行家嗅到服务费、廉价债务资金推波助澜。

[[伯克希尔哈撒韦]]以全股权基础评估收购，从不指望"协同效应"。对[[杠杆]]的厌恶确实拖累了短期回报，但[[芒格]]和巴菲特睡得很踏实。

年内完成的重要交易：收购飞行员旅行中心38.6%权益（年销售额200亿美元的卡车旅行中心龙头）；[[克莱顿房屋]]收购两家传统住宅建筑商；肖氏工业收购美国地板公司；家庭服务公司收购多家房产经纪公司，经纪人总数达40,950人。

### 保险业务的核心地位

1967年以860万美元收购[[国民保险公司]]，获得670万美元净有形资产和1,940万美元[[保险浮存金]]。从那以后，[[保险浮存金]]对[[伯克希尔哈撒韦]]至关重要。

[[保险浮存金]]产生方式：保费预收而损失后发；某些损失需多年才浮出水面；部分赔付延续数十年。[[伯克希尔哈撒韦]]专注长尾业务，2017年[[保险浮存金]]达1,145亿美元，远超所有同行。

与银行存款不同，财产意外险[[保险浮存金]]不能被提取，不会遭遇"挤兑"。这是[[伯克希尔哈撒韦]]最看重的特质，使其能从容应对各种经济冲击。
