# 2007 巴菲特致股东信

## 核心要点

- [[伯克希尔哈撒韦]]净资产增加123亿美元，每股账面价值增长11%，43年复合增长率达21.1%
- 房地产泡沫破裂暴露了金融机构"放松的放贷标准"问题——"只有退潮的时候，你才知道谁在裸泳"
- 真正伟大的企业必须拥有持久的[[护城河]]，能以极少的资本投入获得丰厚回报
- 完成收购马蒙集团60%股份，这是[[伯克希尔哈撒韦]]历史上最大的现金收购
- [[保险业]]盛宴已结束，2008年利润率将显著下滑

## 详细摘要

### 年度业绩与市场环境

2007年[[伯克希尔哈撒韦]]76家运营业务总体表现良好，少数与房地产相关的业务（砖块、地毯、房地产经纪）受到轻微冲击。与此同时，大型金融机构因深陷"放松的放贷标准"而遭遇触目惊心的问题。[[富国银行]]CEO对此评论道："旧的赔钱方法还好端端的，这个行业偏要发明新的赔钱方法。"

房价永远上涨的信念让借款人的收入和自有资金在放贷者眼中变得无足轻重。如今国家正为这种错误信念饱受煎熬。

### 价值评估的两把标尺

[[伯克希尔哈撒韦]]的价值主要源于两大板块：

第一块是投资组合，年底总计1,410亿美元。[[保险浮存金]]为投资提供了590亿美元的资金来源。只要保险承保能做到盈亏平衡，浮存金就是"免费"的。

第二块是非保险业务收益。过去二十年来，公司越来越重视发展这部分收益。每股税前收益从1965年的4美元增长到2007年的4,093美元，整个期间年复合增长率为17.8%。

### 独特的管理架构

[[伯克希尔哈撒韦]]的经理人群体非同寻常。他们大多数在经济上根本不需要工作，继续经营纯粹因为乐在其中。他们衡量成功的标准不是能否坐上更高的位子，而是企业的长期业绩表现。这种"今天在这里，永远在这里"的心态，赋予了公司真正的竞争优势。

### 重大收购

圣诞节完成了收购马蒙集团60%股份的交易，初始支出45亿美元。马蒙拥有70亿美元销售额和约2万名员工，最大业务是联合槽罐车公司。这笔交易的种子早在1954年就播下了，当时年轻的[[巴菲特]]被[[格雷厄姆]]派去参加洛克伍德巧克力公司的股东大会，在那里结识了杰伊·普利兹克。

交易方式正合杰伊的心意：仅凭财务报表确定价格，没有聘请顾问，也没有斤斤计较。一言为定。

### 什么是伟大的企业

[[芒格]]和[[巴菲特]]寻找的公司需要满足：能理解的业务、良好的长期经济前景、有能力且值得信赖的管理层、合理的[[买入价格]]。

真正伟大的企业必须拥有持久的[[护城河]]——比如成为低成本生产商（盖可保险、好市多），或拥有强大的全球[[品牌]]（[[可口可乐]]、[[吉列]]、美国运通）。"持久性"标准排除了处在快速变化行业中的公司，也排除了成功依赖某位伟大管理者的企业。

[[喜诗糖果]]是梦幻般生意的原型。1972年以2,500万美元买下时，税前利润不到500万美元，运营资本仅需800万美元，投入资本税前回报率高达60%。去年税前利润达8,200万美元，而自1972年以来仅需再投入3,200万美元。税前利润总计达到13.5亿美元，几乎全部回流到[[伯克希尔哈撒韦]]用于收购其他企业。

## 原文金句

> "只有退潮的时候，你才知道谁在裸泳——而我们在一些最大金融机构所目睹的景象，实在是惨不忍睹。"

> "一家真正伟大的企业必须拥有一条持久的'护城河'，保护其投入资本获得丰厚回报。"

> "一条需要不断重建的护城河，最终等于没有护城河。"

> "拥有'希望之钻'的一小部分权益，也远胜过拥有一整颗假钻石。"

> "没有哪条规矩说你必须把钱投回赚到钱的地方。事实上，这样做通常是个错误。"
