# 2010 巴菲特致股东信

## 核心要点

- [[BNSF铁路]]收购效果超预期，使[[伯克希尔哈撒韦]]"常规"盈利能力税前增长近40%，税后增长超30%
- 铁路相比卡车运输具有三倍燃油效率优势，每吨货物一加仑柴油可跑500英里
- 公司估算当前资产的常规盈利能力约为税前170亿美元、税后120亿美元
- 46年来每股账面价值年复合增长率约20.2%，从19美元增长至95,453美元
- 对美国未来保持坚定乐观，2011年计划创下80亿美元资本支出新纪录

## 详细摘要

### 年度业绩与铁路收购

2010年[[伯克希尔哈撒韦]]每股账面价值增长13%。年度最大亮点是完成[[BNSF铁路]]收购，交易效果超出预期。这笔收购使公司股份增加6%，动用220亿美元现金，但现金很快得到补充，经济效益非常出色。

[[BNSF铁路]]相比卡车运输拥有巨大的成本和环保优势。去年创下新纪录：每运输一吨货物，一加仑柴油能跑500英里，燃油效率是卡车的三倍。这意味着更低的运营成本、更少的温室气体排放和更低的进口石油需求。

### 对美国的信心

在人们对美国经济普遍悲观时，[[伯克希尔哈撒韦]]用行动展示了投资热情——2010年在不动产和设备上花了60亿美元，其中90%投在美国境内。2011年计划创下80亿美元资本支出新纪录，增加的20亿美元将全部投在美国。

资本总是流向机会，而美国遍地是机会。无论今天看起来多么风平浪静，明天总是充满不确定性——但不应被此吓住。与巴菲特出生时相比，美国人的生活水平已提高六倍。释放人类潜能的美国制度至今仍充满活力。

### [[内在价值]]的三大支柱

[[内在价值]]有三个关键组成部分：

**第一，投资组合**：年末股票、债券和现金等价物市值合计1,580亿美元。[[保险浮存金]]提供了660亿美元资金来源。只要保险[[承保纪律]]能盈亏平衡，这些钱就是"免费"的。

**第二，非保险业务收益**：来自68家非保险公司。过去二十年，公司越来越重视拓展这部分收益。40年来非保险业务每股税前收益年复合增长率达21.0%。

**第三，[[留存收益]]的使用效率**：这是更主观的因素。有些公司能把留存的每一美元变成两美元，有些只能变成五毛钱。1960年代末，交给山姆·沃尔顿手里的每一美元，与交给西尔斯百货CEO手里的每一美元，命运截然不同。

### 竞争优势

**管理团队**：拥有一批真正出色的管理者，对业务和[[伯克希尔哈撒韦]]都有非同寻常的忠诚。很多CEO本身已经很富有，工作纯粹因为热爱。他们不用参加总部会议，不用操心融资，也不用应付华尔街骚扰。"招好人，少管理"这条准则对所有人都适用。

**资本配置自由**：大多数公司只在自己一直经营的行业里再投资，被限制在一个小天地里。[[伯克希尔哈撒韦]]配置资本时不受任何制度性约束，可以把任何机会与一大堆其他机会进行比较，还会拿它和有价证券市场上的机会进行比较。
