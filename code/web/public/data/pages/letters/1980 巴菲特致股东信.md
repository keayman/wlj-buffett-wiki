# 1980 巴菲特致股东信

## 核心要点

- 经营收益增长至4,190万美元，但股本回报率从18.6%下滑至17.8%
- 持股低于20%的公司留存收益不计入报表，但这些"隐藏收益"实际上超过了全年报告经营收益
- 高通胀相当于对资本额外征税，使大多数企业投资无法为股东创造正的实际回报
- 回购自家股票是留存收益最有吸引力的用途之一
- 16年来每股账面价值年复合增长率达20.5%

## 详细摘要

### 无控制权持股的会计处理与真实价值

[[伯克希尔哈撒韦]]将大量资源集中于[[保险业]]，资产大量配置在持股低于20%的公司。按会计准则，这类公司只有分红才计入收益，[[留存收益]]被完全忽略。然而这些公司保留的属于"我们"的收益，实际上超过了全年报告经营收益总额——收益"冰山"只有不到一半浮出水面。

留存收益的价值不取决于持股比例，而取决于资金被用在何处以及产生的回报水平。"重要的是行为本身，而非行为者。"如果一棵树生长在部分拥有的森林里，即便财务报表未记录其生长，我们依然拥有那棵树的一部分。

### 回购：最确定的资本运用

在被投资公司使用[[留存收益]]的方式中，[[回购]]自家股票格外令人兴奋。如果优秀企业以远低于[[内在价值]]的价格出售，还有什么比以便宜价格扩大股东权益更确定、更有利可图的资本运用？证券市场的竞价机制让公司有机会以协商收购所需价格50%的代价买回自家业务。

### 长期绩效与复利效应

自现任管理层接掌16年来，每股账面价值从19.46美元增长至400.80美元，年[[复利]]增长率为20.5%。长期回报远超每年报告收益，主要原因是无控制权持股的留存收益最终转化为市场价值增长。这种转化高度不均匀、时间无法预测，但只要[[买入价格]]合理，市场几乎必然会认可留存收益的累积效果。

### 通胀：股东的隐形杀手

高通胀相当于对资本额外征税。在12%通胀环境下，即便股本回报率达20%的企业将全部收益分配给50%税率的股东，实际上也在蚕食其真实资本。通胀税不受报告收益约束，足以把大多数企业的正回报变成全体股东的负回报。

适用中高税率的个人投资者不应指望从一般美国企业获得任何实际长期回报——企业平均股本回报率已被通胀隐性税和显性税完全抵销。[[伯克希尔哈撒韦]]对此没有灵丹妙药。

### 主要业务表现

保险集团承保收益674万美元，净投资收益3,094万美元。[[喜诗糖果]]税前收益1,503万美元。[[布法罗新闻报]]仍亏损281万美元。伊利诺伊国民银行贡献532万美元。
